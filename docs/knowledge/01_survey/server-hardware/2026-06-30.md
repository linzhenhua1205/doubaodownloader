## 1️⃣ 🔥 AMD EPYC 8005 "Sorano" 发布：SP6插槽史上最大飞跃 — 84核Zen 5 + DDR5-6400

**来源**: [ServeTheHome - AMD EPYC 8005 Sorano Completely Changes AMD SP6](https://www.servethehome.com/amd-epyc-8005-sorano-completely-changes-amd-sp6/) (Patrick Kennedy, June 29, 2026)

### 核心规格

| 参数 | EPYC 8005 Sorano | EPYC 8004 Siena (上一代) | 变化幅度 |
|:-----|:----------------|:------------------------|:---------|
| **架构** | Zen 5 (全缓存) | Zen 4c (密集核心) | **核心架构换代** |
| **最大核心** | **84核 / 168线程** | 64核 / 128线程 | +31% |
| **L3缓存** | **384MB** | 128MB | +200% |
| **内存** | **DDR5-6400** | DDR5-4800 | +33% 频率 |
| **CXL** | **CXL 2.0** | CXL 1.1+ | 代际升级 |
| **TDP范围** | 225W (默认) | 200W (默认) | +25W (+12.5%) |
| **系统级功耗增量** | — | — | **<10%** (约30-34W) |
| **最大频率** | **大幅提升** (Zen 5架构) | 较低 (Zen 4c密集核心) | 显著 |
| **封装** | SP6 (同EPYC 8004) | SP6 | **引脚兼容** (需BIOS更新) |

### SKU堆栈与定价

| SKU | 核心/线程 | TDP | 价格(USD) |
|:----|:---------|:----|:---------|
| EPYC 8005 (顶配84核) | 84C/168T | ~225W | $5,799 |
| EPYC 8005 (64核) | 64C/128T | ~225W | ~$5,000 (84核仅略贵) |
| EPYC 8005 入门 | 8C/16T | ~225W | $529 |
| **价格跨度** | 8→84核 | 225W | **$529 - $5,799** |

### 技术分析 — 战略意义

1. **同一插槽, 完全不同的产品**：EPYC 8005 与 9005 "Turin" 和 4005 系列使用相同架构的 **全缓存 Zen 5 核心**（非 Zen 5c 密集核心）。这意味 AMD 在 SP6 这条中端线上提供了与旗舰同级的单线程性能。

2. **Zen 4c → Zen 5 的质变**：
   - 上代 8004 用 Zen 4c（密集核心），L3 仅 128MB，频率受限
   - 8005 用全规格 Zen 5，L3 暴增到 384MB（+200%）
   - 对内存敏感型工作负载（数据库、AI推理、HPC）有巨大优势

3. **功耗效率**：虽然 TDP 从 200W→225W（+12.5%），但实际系统级功耗增量仅 <10%（~30-34W），**核心数+31%、缓存+200%、内存频率+33%**，每瓦性能大幅提升。

4. **市场定位**：AMD 在 SP6 上用全规格核心而非密集核心，可能是应客户需求——客户不希望在中端平台上妥协单线程性能。使 SP6（通常用于边缘/电信/中小型部署）获得与企业旗舰几乎相同的核心架构。

5. **与竞品对比**：SP6 现在有 84 核 Zen 5，而 Intel 的 Granite Rapids-D（Xeon 6 SoC）最大 38核，在核密度上 AMD 仍然领先。

## 2️⃣ 🔥 Supermicro GB300 Super AI Station 深度解析 — DGX Station 内部的秘密

**来源**: [ServeTheHome - Taking an Up-Close Look at the Supermicro GB300 Super AI Station](https://www.servethehome.com/taking-an-up-close-look-at-the-supermicro-gb300-super-ai-station/) (Ryan Smith, June 27, 2026)

### 系统规格一览

| 参数 | 规格 |
|:-----|:-----|
| **SoC** | NVIDIA Grace Blackwell GB300 (72核Grace CPU + Blackwell Ultra GPU) |
| **CPU内存** | 496GB LPDDR5X SOCAMM (4×模块, 液冷覆盖) |
| **GPU内存** | **252GB HBM3e** (7/8堆栈启用, 降额) |
| **GPU内存带宽** | **7.1 TB/s** |
| **TDP** | **1,600W** (整机) |
| **冷却** | **全液冷** — 冷板覆盖CPU/GPU/内存, 前置散热排+3风扇 |
| **网络** | NVIDIA ConnectX-8 **2×400GbE** (QSFP112) |
| **存储** | 4× PCIe Gen5 x4 M.2 2280 (演示用 480GB Micron 7450 Gen4) |
| **扩展槽** | 2× PCIe x16 (x8电) |
| **附加GPU** | 可选 NVIDIA RTX PRO (Blackwell 2000/4000/6000) — GB300本身无图形输出 |
| **售价** | ~**$125,000** |
| **尺寸** | 塔式工作站, 可配机架安装套件 → **5U 服务器** |
| **操作系统** | DGX OS (Linux) 或 DGX Station for Windows (2026 Q4) |
| **OEM伙伴** | ASUS, Dell, Gigabyte, HP, MSI, **Supermicro** |

### 关键工程细节

**1. GB300 降额设计**
- DGX Station 中使用的 GB300 仅启用了 **7/8 个 HBM3e 堆栈**（共252GB而非完整的288GB）
- 这是为塔式/桌面部署做热管理折中——1,600W 在 5U 空间已经极度紧张
- 7.1 TB/s 带宽仍是桌面平台顶级

**2. 液冷架构**
- **冷板覆盖**: GB300 SoC + 4个 LPDDR5X SOCAMM 内存模块
- **前置散热排**: 3个风扇推气通过散热排 → 整机从前到后风道
- 前→后风道设计同时支持塔式与 **5U 机架安装**（双重用途的区分优势）

**3. 无图形输出 — 需额外GPU**
- GB300 是纯计算优化设计，**无显示输出**
- 必须安装 RTX PRO 卡才能获得 BMC 以上级别的图形
- 这是 DGX Station 区别于普通工作站的特性

**4. 网络能力**
- 2× 400GbE (ConnectX-8) = **800Gbps 总网络带宽**
- 支持多台 DGX Station 组成小集群，或作为高速回传链路
- 相比数万美元的数据中心级 800G 网卡，这是桌面系统的顶级网络性能

**5. 供电要求**
- 1,600W 负载需求 **专用供电回路**
- 不可用普通办公插座，需要数据中心级电力规划

### 市场意义

NVIDIA 正在将旗舰级 AI 算力从数据中心推向桌面。DGX Station for Windows (2026 Q4) 将进一步拓展到 Windows 生态的 AI 开发者市场。$125,000 的定价使其成为**「个人超算」的终极形态**。

## 3️⃣ 🏗️ Wiwynn × TE Connectivity：800V DC 液冷汇流排 — 下一代 AI 数据中心供电方案

**来源**: [ServeTheHome - Liquid-Cooling a TE Connectivity 800V DC Busbar and More from the Wiwynn Booth](https://www.servethehome.com/liquid-cooling-a-te-connectivity-800v-dc-busbar-and-more-from-the-wiwynn-booth/) (Patrick Kennedy, June 26, 2026)

### 核心创新

| 技术 | 详情 |
|:----|:-----|
| **供电电压** | **800V DC HVDC** (高压直流) — 相对传统 48V/400V 的跨越式升级 |
| **电缆组件** | TE Connectivity HVDC IT GEAR → **ELCON MICRO+** 电源线组件 |
| **转换** | 800V/75A IT端 → 400V ELCON MICRO+ 服务器端 |
| **冷却** | **汇流排本身也液冷** — 液冷块覆盖汇流排+加速器 |
| **机架缩放网络** | 新一代液冷接口+机架内缩放网络连接器 |
| **制造工艺** | **机器人装配设计** — 整条电源线组件在制造阶段一次性插入 |

### 技术背景

**为什么需要 800V DC？**

1. **AI加速器功耗暴增**：下一代 AI 加速器每个模块功耗超过 1000W（图中演示了 100+ cm² 的加速器dummy模块）
2. **ORv3 机架已到极限**：现有 Open Rack v3 标准的 48V/400V 供电在 >50kW/rack 的密度下效率骤降
3. **电流管理**：更高电压 = 更低电流 = 更少 I²R 铜损 = 更细的电缆

**示范系统架构**:
```
ORv3 机架节点
├── 管理模块 (左侧)
├── 4× E1.S SSD (前端)
├── 光网络笼 (中央, dummy)
├── 2× 下一代 AI 加速器 dummy 模块 (100+ cm²)
├── 800V DC 供电线束 (穿过加速器之间)
│   ├── TE HVDC IT GEAR 端 ← 800V/75A 来自机架汇流排
│   └── ELCON MICRO+ 端 → 400V 至节点内部配电
└── 液冷接口 (后板)
    ├── 液冷喷嘴
    └── 机架内缩放网络连接器
```

### 液冷汇流排的战略意义

- **汇流排也液冷**意味着：不仅仅是芯片在发热，**配电系统本身也在发热**
- 800V DC 高压下的接触电阻和 I²R 损耗会产生可观热量，需要主动冷却
- 这是数据中心供电架构从「电气工程」向「热管理工程」扩展的标志性信号

### 供应链与制造影响

- 整条电源线组件设计为 **drop-in（一次性插入）**，适合机器人自动装配
- 手工装配 800V 高压线束在生产线上不现实（安全+一致性）
- 推动服务器制造向 **高度自动化装配** 转型

### 时间线与展望

| 阶段 | 状态 |
|:----|:-----|
| 概念验证 | ✅ Wiwynn Computex 2026 实物演示 |
| 标准化 | 预计 ODCC/OCP 跟进 800V DC 标准制定 |
| 商用部署 | 预计下一代 AI 加速器（~2027+）将大规模采用 |
