## 1️⃣ NVIDIA Vera Rubin 进入量产，DGX Station for Windows 发布

**来源**: ServeTheHome - Ryan Smith (6/1); DIGITIMES Most-Read (6/9)

### Vera Rubin 平台量产
- NVIDIA Computex 2026 主题演讲确认 Vera Rubin 平台已进入 **全面量产**
- **Vera CPU**: 88核设计，基于 NVIDIA 自研 **Olympus CPU 核**
- **GPU**: Rubin 架构新 GPU
- **工艺**: 均采用 **TSMC 3nm**（比 Grace Blackwell 的 N4P 先进一代）
- **出货**: 2026年 **秋季** 开始发货
- **Samsung 已出货 HBM4 用于 Vera Rubin**，并正在与 NVIDIA 谈判 **HBM5 及下一代代工合作**（DIGITIMES 6/9 最热文章）

### DGX Station for Windows
- **SoC**: NVIDIA Grace（72核 Neoverse V2 CPU）+ **Blackwell Ultra GPU**
- **GPU 显存**: 252GB HBM3e
- **CPU 内存**: 496GB LPDDR5X SOCAMM
- **功耗**: **1600W** 整机功耗
- **网络**: ConnectX-8 800Gbps 以太网 + 10GbE
- **上市**: **2026 Q4**
- **OEM 合作**: ASUS、Dell、Gigabyte、HP、MSI、Supermicro 六家均计划推出

## 2️⃣ Intel Xeon 6+ Clearwater Forest 深度解析：288 核 E-core 正式出货

**来源**: ServeTheHome - Patrick Kennedy (6/1); Tom's Hardware CPU 栏目 (6/8)

### 核心规格
| 参数 | 规格 |
|:-----|:-----|
| 核心数 | 最高 **288 E-cores**（无超线程） |
| 末级缓存 | **576 MB** LLC（12个计算芯粒 × 48MB/块） |
| 内存 | 12通道 **DDR5-8000**，最大 1.5TB（12×128GB RDIMM） |
| 工艺 | **Intel 18A**（PowerVia 背面供电 + RibbonFET GAA） |
| 封装 | 12计算芯粒 + 3有源基底 + 2 I/O 芯粒 + 12 EMIB |
| TDP | **450W**（< 2W/核） |
| 平台兼容 | 仅 **BIOS 更新**即可在现有 Xeon 6900P 平台使用 |

### 性能对比（Intel 官方数据）
- **vs Xeon 6780E**（144核 Sierra Forest）: 2.26× 平均性能 / 1.55× perf/watt
- **vs EPYC 9965**（192核/384线程 500W）: 宣称 **1.3× 平均每线程性能** / 1.3× perf/w/thread
- **vs Sierra Forest 电信场景**: +30% 同核数性能 / +60% perf/watt / 机架功耗 -38%
- **vs Cascade Lake（2代Xeon）**: 1机架 ≈ 48机架（960台旧服务器）的整合比

### 关注点
- **Diamond Rapids（P-core）延期至2027**，E-core 路线加速
- Xeon 6900E+ 系列支持 **2DPC**（但还没有对应主板平台）
- **Intel SGX + TDX** 机密计算
- **Application Energy Telemetry（AET）**: 按应用级核心能耗数据做负载调度

## 3️⃣ Gigabyte 40节点/1U 超密度集群 R1C7-K0A-AS1

**来源**: ServeTheHome - Cliff Robinson (6/6)

### 极致密度设计
- **40 节点**部署在 **1U** 机箱内
- 每节点: Intel Core Ultra 7 258V（**Lunar Lake**，4P+4E=8核/节点）、32GB LPDDR5X、2× M.2 SSD
- 总计: **320 核 + 1.28TB 内存 + 80 SSD + 40 iGPU** / 1U
- **5 个可插拔舱**（每舱8节点，前3后2排列），类似 GPU 卡槽设计

### 网络与供电
- 2× **3.2kW 钛金级** 电源（总计 6.4kW/1U）
- 2× **QSFP28** 后出端口（内部互联方式未公开，推测为以太网交换芯片整合）
- 内置 **机箱管理控制器** + 管理网口

### 扩展推论
- 40U 机架可容纳: **12,800 CPU 核 + 3,200 SSD + 1,600 iGPU + 51.2TB 内存**
- 仅需 80 电源线 + 80 100GbE 端口 + 40 管理端口
- 尚无 ECC 支持（Lunar Lake 片上内存未提供 ECC 配置）
- **尚未正式发布**

## 4️⃣ RTX Spark SFF Mini-PC 四厂商详评（Computex 2026）

**来源**: ServeTheHome - Ryan Smith (6/5)

### 四款机型对比
| 品牌 | 型号 | 尺寸 | 特点 |
|:-----|:-----|:-----|:------|
| **ASUS** | ProArt GA10 | 150×150×51mm | 140W 散热设计，与Ascent GX10 同模具 |
| **Dell** | XPS RTX Spark Desktop | 比 GB10 更高 | XPS 品牌，全新设计 |
| **Lenovo** | SFF RTX Spark | — | 新机箱（非 ThinkStation PGX 复用） |
| **MSI** | EdgeMesa N AI+ | — | 白色机箱，立式设计 |

### 共同规格
- **SoC**: NVIDIA N1X（GB10 重新贴标），20 CPU 核 + 48 SM Blackwell iGPU
- **内存**: 最高 **128GB LPDDR5X** 统一内存
- **存储**: PCIe Gen5 ×4 M.2 槽
- **网络**: **10GbE** + Wi-Fi 7 + Bluetooth 5.4（ASUS 确认）
- **I/O**: 4× USB-C 20Gbps + HDMI 输出
- **与 DGX Spark 关键差异**: **没有 ConnectX-7 NIC 和 QSFP 端口**（降低成本）

### 市场定位
- AI 开发者 + 内容创作者
- Windows 平台的 Mac Studio / AMD Ryzen AI Halo 竞品
- **无 ConnectX-7 税**，价格预期低于 DGX Spark

## 5️⃣ DIGITIMES 关键供应链动态（6/9-6/10）

**来源**: DIGITIMES 首页 Most-Read 排行 + 实时新闻

### AI 服务器供应链核心转移
- **AI 供应链短缺从芯片转向设备**: 不再是 GPU 短缺，而是 **封装设备、测试设备、PCB 产能** 成为新瓶颈
- **AI 机架功率密度激增催生供电架构升级**: 单机架功率从 40kW → 100kW+，推动 **HVDC 800V + BBU + 备用电源** 全面升级
- **Samsung Foundry 转向 5nm/8nm 订单**: 在 2nm 反攻前先巩固成熟制程
- **AMD 2nm 转单三星**: 冲击 TSMC AI 供应链独占性
- **SK 海力士加速龙仁工厂建设**: DRAM 产能军备竞赛

### 服务器出货强劲
- **台湾服务器制造商 5 月营收强劲**: AI 需求持续拉动
- **全球半导体市场 2026 年 $1.5 万亿**: 内存暴增 250%（WSTS 预测）
- **ASICs 2027 年可能挑战 NVIDIA GPU 主导**: DIGITIMES 分析师观点

## 📊 综合趋势判断

1. **超节点军备竞赛白热化**: NVIDIA Vera Rubin（88核 CPU + Rubin GPU + HBM4） vs AMD Helios/MI455X（40PFLOPS）局面趋于清晰，2026H2 将迎来首次正面交锋
2. **机架密度极限持续被突破**: Gigabyte 超密度集群（320核/1U）和 RTX Spark SFF 生态（128GB 统一内存/手掌大小）展示了两条截然不同的密度路线
3. **CPU 赛道重燃战火**: Intel Xeon 6+ 288核 E-core vs AMD EPYC Turin（Zen 5）竞争白热化，并且 AI Agent 驱动的推理负载正在成为新的 CPU 需求增长极
4. **供应链从芯片短缺转向设备+PCB 瓶颈**: 这一转变可能影响 2026H2-2027 交付节奏
5. **液冷已成 AI 服务器标配**: 1600W DGX Station 和 6.4kW 1U 集群的数据都指向传统风冷已到极限
