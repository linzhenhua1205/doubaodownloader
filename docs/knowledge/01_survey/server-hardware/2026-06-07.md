## 1️⃣ 🔥 Gigabyte R1C7-K0A-AS1 — 40节点/1U超密度集群

**来源**: ServeTheHome, Cliff Robinson, 2026-06-06
**相关**: COMPUTEX 2026 现场直击

| 项目 | 规格 |
|:-----|:------|
| 节点数 | **40节点 / 1U** |
| CPU | 40× Intel Core Ultra 7 **258V** (Lunar Lake), 4P+8E core/节点 |
| 总核心 | **320核 / 1U**（每节点12核） |
| 内存 | 40× 32GB LPDDR5X = **1.28TB / 1U** |
| 存储 | 40× 2× M.2 = **80块 SSD / 1U** |
| GPU | 40× **Intel Arc iGPU**（集成于 Lunar Lake） |
| 网络 | 2× **QSFP28**（100GbE）后出，通过集线开关整合 |
| 电源 | 2× **3.2kW** Titanium 等级冗余电源 |
| 管理 | 内置 **Chassis Management Controller** |
| 连接 | 5× 8节点滑轨（前3后2），每滑轨 2× MCIO 8i |

**架构解读**:
- 节点以类似GPU卡槽的**滑轨式弹夹**插拔，每滑轨8节点
- 后端 MCIO 交换芯片将40节点聚合为 2× 100GbE 上联
- 3.2kW×2 供电设计说明整体功耗约 5-6kW/1U

**密度对比**（40U 机架）:
> 40节点/1U = 机架满配 1,600 节点 = 12,800 核心 + 3,200 SSD + 1,600 iGPU + 51.2TB LPDDR5X，仅需 80 根 100GbE 线缆 + 40 根管理线

> 比 Arm AGI CPU 密度更高，**AI 推理 / 视频转码 / VDI 场景的密度天花板**

## 2️⃣ 🟢 RTX Spark SFF Mini-PC 四厂商详情

**来源**: ServeTheHome, Ryan Smith, 2026-06-05
**相关**: COMPUTEX 2026, NVIDIA N1X/GB10

### 通用 SoC 规格
| 参数 | 详情 |
|:-----|:------|
| SoC | NVIDIA RTX Spark (N1X), 实为 **GB10 重命名** |
| CPU | 20核 Arm Cortex |
| GPU | **48 SM Blackwell** 集成 GPU |
| 内存 | 最高 **128GB LPDDR5X 统一内存** |
| 网络 | **10GbE** RJ45（**无 ConnectX-7 QSFP**） |
| 无线 | Wi-Fi 7 + Bluetooth 5.4 |
| 存储 | PCIe Gen5 ×4 M.2 |
| USB | 4× USB-C 20Gbps |
| 系统 | **Windows 11**（第一次 NVIDIA Arm SoC 支持 Windows） |

### 四款机型对比

| 厂商 | 型号 | 尺寸 | 散热 | 特色 |
|:-----|:-----|:-----|:-----|:------|
| **ASUS** | ProArt GA10 | 150×150×51mm | 140W 散热设计 | ProArt 创意定位, Kensington锁 |
| **Dell** | XPS RTX Spark Desktop | 高于 GB10 (未公开) | 更高散热 | XPS 品牌, 高于 NVIDIA 标准 |
| **Lenovo** | SFF RTX Spark | 新设计 (未公开) | 新机箱 | ThinkStation PGX 对应 |
| **MSI** | EdgeMesa N AI+ | 白色机箱 (未公开) | EdgeXpert 演变 | AI 开发者/数据科学家定位 |
| **Microsoft** | Surface RTX Spark Dev Box | ~1L 全铝机身 | **100W TDP**, 1000孔通风 | W11 Pro 预装+VS Code+GitHub Copilot |

### 🏆 关键差异化

所有 RTX Spark mini-PC 与 GB10/DGX Spark 的**核心区别**：**去掉了 ConnectX-7 和 QSFP 端口**

- 降成本的主动选择（ConnectX-7 太贵，$800-1200/片）
- Windows 生态定位 vs Linux 多机互联（DGX Spark 主打）
- 目标用户是个人开发者/创作者，不是集群部署

## 5️⃣ 🔧 DIGITIMES 散热/供应链动态

**来源**: DIGITIMES Asia, 2026-06-07

### 5.1 🌡️ Niching 散热片出货大增 — AI 驱动

> Niching Industrial Corp.（**尼西科技**）称 AI 高速计算和数据传输推动芯片功耗和散热需求全球上升，其散热片出货持续攀升。收购 **Ming June Yuan**（明均源）后盈利能力有望改善，市场正转向更大的散热片和更复杂的冷却方案。

### 5.2 💧 Aewin 推动双相液冷 — 高密度 AI 机架

> **Aewin（安勤科技）** 在 COMPUTEX 展示面向高密度 AI 机架的 **双相直接液冷（Two-Phase Direct Liquid Cooling）** 方案，针对游戏和 AI 数据中心场景。双相液冷比单相液冷散热效率更高，预计将成为 2027-2028 年 >500kW/机架的主流方案。

### 5.3 🏭 ASRock Rack 获 587 台 GPU 服务器订单 — 泰国 AI 数据中心

> **ASRock Rack（永擎电子）**，Pegatron（和硕）旗下，获泰国 AI 数据中心 **587 GPU 服务器** 大单。标志台湾 ODM 厂商进一步渗透东南亚 AI 基础设施市场。
