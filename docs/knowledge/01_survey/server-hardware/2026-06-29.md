## 1️⃣ 🔥 Qualcomm Investor Day 2026：Dragonfly C1000 + HBC + Modular 全面出击数据中心

**来源**: [ServeTheHome - Qualcomm Investor Day 2026 Data Center Announcements CPUs, AI Accelerators, and More](https://www.servethehome.com/qualcomm-investor-day-2026-data-center-announcements-cpus-ai-accelerators-and-more/) (Patrick Kennedy, June 24, 2026)

### 核心发布：Qualcomm Dragonfly C1000 CPU

| 规格 | 参数 |
|:-----|:-----|
| **核心数** | 250+ 核心 chiplet 设计 |
| **频率** | **~5GHz**（vs Intel Xeon 6+ Clearwater Forest / AMD EPYC Turin Dense 降频运行策略，这是关键差异化） |
| **互联** | PCIe Gen7, CXL |
| **内存** | LPDDR + 可选 **HBC（High Bandwidth Compute）** 堆叠附件 |
| **RAS** | 企业级可靠性特性 |
| **性能功耗** | 声称 **2× 性能/瓦** 优于竞品 |

**战略意义**: Cristiano Amon 将"**rack-scale 数据中心基础设施**"列为首要未来方向。Qualcomm 表示所有已生产服务器 CPU 正在被一抢而空。

### 核心技术创新：HBC（High Bandwidth Compute）

- **原理**: 将 LPDDR 内存**直接堆叠在计算 Die 上方**，消除 HBM 经过 CoWoS 中介层的**功耗税**
- **声称**: 最佳性能功耗比（"HBC eliminates the HBM tax without SRAM"）
- **路线图**: AI250 代（2027年）→ HBC Gen2（AI300?）将集成 **ESUN + UALink** 互联
- **容量**: AI250 首款 HBC 产品约 **768GB**

### 关键产业合作

| 合作方 | 内容 |
|:-------|:-----|
| **Meta**（Mark Zuckerberg 视频亮相） | 多代处理器协议，覆盖 AI 推理 |
| **Modular（收购）** | 统一软件栈，对标 CUDA/Dynamo/Triton，提供异构计算能力 |
| **Microsoft**（Satya Nadella） | 从 PC 到 AI Agent + HBC 数据中心全方位合作 |
| **Alphawave（收购）** | 补充 die-to-die / CPO / PAM4 SerDes 连接能力 |

## 3️⃣ ⚡ Intel Nova Lake — 52 核 P-Core 旗舰，TDP 高达 474W

**来源**: [Tom's Hardware - Intel's next-gen 52-core Nova Lake CPU could pull up to 474W](https://www.tomshardware.com/pc-components/cpus/intels-next-gen-52-core-nova-lake-cpu-could-pull-up-to-474w) (Kunal Khullar, June 27, 2026)

### 核心参数

| 规格 | 数据 |
|:-----|:-----|
| **核心数** | **52 核**（P-core，推测为高性能核心） |
| **TDP** | **474W**（远超当前 Intel 桌面旗舰，接近服务器级功耗） |
| **架构** | Nova Lake — Intel 下一代客户端/工作站架构，预计 2027-2028 |

### 产业意义

- 474W TDP 标志着**桌面/HEDT CPU 功耗迈入 500W 时代**（i9-14900K 约 253W，翻倍）
- 52 P-core 配置暗示 Intel 正将服务器级核心数下放到高端桌面/工作站，或定位为 **HEDT/工作站旗舰**
- 散热挑战：需配套 360mm+ AIO 液冷或定制水冷方案
- 与 Intel Xeon 6+ (Clearwater Forest) 关系：推测 Nova Lake-EX (Mountain Rapids?) 将成为数据中心版本

> ⚠️ 该信息来自Tom's Hardware报道，尚未获得 Intel 官方确认，属工程样品/泄露信息。

## 4️⃣ 🇨🇳 Loongson 3C3000 — 国产 16 核 LoongArch 服务器 CPU

**来源**: [Tom's Hardware - China's Loongson launches homegrown 16-core 3C3000 server CPU built on LoongArch](https://www.tomshardware.com/pc-components/cpus/chinas-loongson-launches-homegrown-16-core-3c3000-server-cpu-built-on-loongarch) (Etiido Uko, June 28, 2026)

### 规格参数

| 规格 | 数据 |
|:-----|:-----|
| **核心** | 16 核，自主 **LoongArch** 指令集架构 |
| **内存** | DDR4 ECC |
| **PCIe** | 32 条 PCIe 通道 |
| **典型功耗** | **40W**（远低于 x86 竞品） |
| **性能** | 号称匹配前代 **3C5000** |
| **连接** | **2× 1GbE** 网口 onboard |

### 产业意义

- 中国半导体自主化持续推进，LoongArch 是完全自研 ISA（非 ARM/x86 授权）
- 16核/40W 定位**低功耗边缘/轻量服务器**场景，性能大致相当于中端 ARM 服务器 SoC
- 32 PCIe lanes 对于服务器级 CPU 偏少（AMD EPYC 有 128+ lanes），限定了其应用场景为**轻量/单路**部署

## 📋 本周综合趋势总结

1. **后NVIDIA时代的多极格局**: Qualcomm HBC+Modular、OpenAI+Broadcom、AMD+Intel 都在构建**异构AI推理**方案，NVIDIA 正面临史上最严峻的生态挑战
2. **互联层面竞争白热化**: UALink、CXL、PCIe Gen7、NVLink 五路互联方案并存，谁能成为 **scale-up 互联标准** 将决定未来5年数据中心架构
3. **功耗密度加速走高**: Nova Lake 474W、GB300 DGX 1600W、HPC机架 1.6MW → **液冷**不再是可选项，而是强制要求
4. **供应链是隐形瓶颈**: PCB/CCL 产能紧张叠加 ABF 载板短缺，AI 服务器实际出货量受限于上游材料
5. **中国自主CPU缓慢前进**: Loongson 3C3000 16核40W 定位轻量场景，与 x86/Arm 高性能差距仍然巨大
