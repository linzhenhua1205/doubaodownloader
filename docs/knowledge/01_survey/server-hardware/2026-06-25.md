## 1️⃣ 🔥 Qualcomm Dragonfly C1000：高通行列数据中心CPU

**来源**: [ServeTheHome - Qualcomm Investor Day 2026 Data Center Announcements](https://www.servethehome.com/qualcomm-investor-day-2026-data-center-announcements-cpus-ai-accelerators-and-more/)

**Qualcomm Investor Day 2026（纽约）** 数据中心全面战略发布，核心要点：

### Qualcomm Dragonfly C1000 CPU
- **架构**: Chiplet设计，**250+ cores**，PCIe Gen7，CXL，LPDDR内存，可选HBC attach，企业级RAS
- **关键差异化**: 核心运行在**~5GHz**高频 — Intel Clearwater Forest / AMD EPYC Turin Dense 均无法达到此频率
- **宣称2× better perf/watt**（高频下实现更优能效，反直觉但吸引人）
- **时间线**: 需PCIe Gen7时代才上市（意味着类似Intel Diamond Rapids / AMD Venice之后的时段）

### High Bandwidth Compute (HBC) — 架构级创新
- 将**内存直接堆叠在计算tile之上**，而非HBM通过CoWoS中介层连接
- 声称消除HBM Tax（中介层数据传输功耗），提供接近SRAM的延迟
- **AI250代（2027）** 首次引入HBC
- **HBC Gen2（AI300?）** 将集成ESUN和UALink互联

### 生态与路线图
- **收购Modular**（Tim Davis联合创始人上台）— 对标CUDA/Dynamo/Triton的软件栈，实现异构计算
- **Meta（Mark Zuckerberg视频）** 宣布多代处理器合作协议 — 继Arm AGI CPU后第二个重要CPU合作伙伴
- **四产品线**: CPU + AI加速器 + 定制硅 + 连接性
- **连接路线图**: Die-to-Die、Co-Packaged Optics、PAM4电/光SerDes、QAM16相干光 — 与Broadcom/Marvell正面竞争
- **Alphawave收购**: 强化SerDes和高速互联IP

### 解读
Qualcomm这次不是Centriq 2400的翻版 — 这次聚焦少数超大规模客户，走定制硅路线。HBC是真正的架构创新（类似AMD 3D V-Cache但更激进）。软件栈仍是最大不确定因素，Modular收购是重点。

## 3️⃣ 🇨🇳 LineShine LX2：首台纯CPU超算登顶TOP500 #1

**来源**: [ServeTheHome - Arm CPUs Take #1 in Top500](https://www.servethehome.com/arm-cpus-take-number-1-in-latest-top500-list-with-chinese-lineshine/)

### LX2 CPU 架构
- **Huawei设计**（推测），单芯片 = 双die
- 每die = **152 Armv9 cores**（含SVE+SME向量/矩阵加速），4个NUMA域 × 38 cores
- 每die: **4GB HBM + 32GB DDR5**（可配Flat拓扑或HBM→DDR缓存拓扑）
- **CPU总计**: 304 cores + 32GB HBM + 256GB DDR5
- **690W TDP** — GPU级功耗
- **800Gbps NICs on-chip**（集成网络）

### 系统规模
- 2 CPU/节点 → 8 节点/刀片(16CPU) → 16刀片/计算框(256CPU) → 2框/机柜(512CPU)
- **~89-90个计算机柜** → 约 **1,379万cores**
- **2.2 EFLOPS**实测（80%效率，vs美国超算50-65%）
- **42MW**功耗，**52 GFLOPs/watt**
- **LingQi 1.6Tbps网络**，4层Fat-Tree + 1光层

### TOP500格局变动
| 排名 | 系统 | 处理器 | 性能 |
|:----:|:-----|:-------|:----:|
| #1 | LineShine | Arm LX2 (纯CPU) | 2.2 EF |
| #2 | El Capitan | AMD MI300A | ~1.74 EF |
| #3 | Frontier | AMD MI250X | ~1.69 EF |
| #4 | Aurora | Intel GPU | ~1.01 EF |

- AMD 占据TOP10中**4席**，NVIDIA Hopper 3席
- **Arm首次登顶** — 纯CPU超算超越GPU超算
- **8年内从Fugaku (Armv8) → LineShine (Armv9) 完成迭代**
