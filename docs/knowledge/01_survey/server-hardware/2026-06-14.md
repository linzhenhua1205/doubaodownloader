## 1️⃣ **Senao SX906 Intel Xeon 6 SoC DPU — 将服务器 CPU 塞进 PCIe 卡** 🔥 **NEW**

- **来源**: ServeTheHome (Patrick Kennedy, Jun 12, 2026)
- **来源链接**: https://www.servethehome.com/this-is-an-intel-xeon-6-soc-dpu-on-a-pcie-card-from-senao-at-computex-2026/

**技术详情**:
- **Senao SX906** 是一款基于 **Intel Xeon 6 SoC (Granite Rapids-D)** 的 DPU（数据处理单元），以标准 PCIe Gen5 卡形态存在
- **CPU 规格**: 三个 SKU 覆盖 **24 / 36 / 38 P-cores**

| 型号 | 核心 | 基础频率 | 总带宽 | TDP |
|:-----|:----|:---------|:-------|:----|
| Xeon 6523P-B | 24C | 2.5GHz | 100G | 295W |
| Xeon 6553P-B | 36C | 2.6GHz | 200G | 355W |
| Xeon 6563P-B | 38C | 2.4GHz | 200G | 355W |

- **网络**: 双 **100GbE** QSFP28（2×100G），Intel E830 控制器提供 200GbE 主机连接
- **PCIe**: PCIe Gen5 **×8 边缘连接器 + 2× ×8 MCIO 扩展**（共 24 条 PCIe Gen5 通道）
- **管理**: **ASPEED AST2600 BMC + AST1060 PFR 控制器**，支持 OpenBMC、Intel PFR 平台固件弹性、TPM 2.0、Secure Boot
- **安全**: Intel Platform Firmware Resilience (PFR)
- **内存**: **4 通道** DDR5
- **特色功能**: 所有 SKU 集成 **QuickAssist (QAT) 加速器**；36/38C SKU 额外集成 **媒体转码加速器**
- **散热**: 双槽主动风冷
- **外形**: FH10.5L+（266mm），双槽，约 1kg
- **供电**: PCIe 边缘 + 16-pin 12VHPWR 辅助供电

**战略意义**:
- 这是目前所见 **最强大的 PCIe 卡形态 DPU**，本质上是"卡上服务器"
- 使用标准的 **x86 P-core 架构**，兼容现有 x86 软件生态，**不需要像 NVIDIA BlueField 那样适配 Arm 或自定义 SDK**
- 由于搭载完整 BMC (AST2600)，可独立管理运行，是真正意义上的 **"服务器 on a card"**
- 竞争对手对标：NVIDIA BlueField-3/4 DPU、AMD Pensando — 但 Senao 使用完整 x86 SoC 提供更高计算能力

## 2️⃣ **Wiwynn 展示 NVIDIA SCADA 存储服务器 — 2.9PB、528M IOPS、GPU 加速存储** 🔥 **NEW**

- **来源**: Tom's Hardware (Anton Shilov, Jun 12, 2026) — 会员专享
- **来源链接**: https://www.tomshardware.com/pc-components/ssds/nvidias-high-speed-ai-data-center-storage-servers-break-cover-touting-2-9-petabytes-of-storage-and-extreme-pcie-6-0-performance-wiwynn-shows-off-scada-server-with-gpu-accelerated-storage

**架构详情**:

| 维度 | 规格 |
|:----|:------|
| 形态 | **NVIDIA MGX** 兼容，**6RU** 机架式 |
| 存储 | 最高 **96 个液冷 E3.S SSD** |
| 容量 | 使用 **96×30.72TB Micron 9650 Pro** (PCIe 6.0) → **2.949 PB** |
| 随机性能 | **5.28 亿 4K IOPS**（聚合随机读） |
| 顺序性能 | 受限于 PCIe Switch/网卡（超过 SSD 本身） |
| GPU | **4× RTX Pro 6000 Blackwell**（作为存储处理器） |
| CPU | **NVIDIA Vera CPU** |
| 互联 | **4× PCIe 6.x Switch** + **4× ConnectX-9 SuperNIC** |
| 功耗 | 最高 **9 kW** |
| 散热 | **全液冷** — SSD 由 6 个独立冷板模组冷却 |
| 定位 | **Tier 3.5** 存储（介于本地 SSD / Tier 4 远程 HDD 之间）|

**架构创新 — GPU 加速存储**:
- **NVIDIA SCADA** (SCaled Accelerated Data Access) 的核心创新是 **让 GPU 直接发起和控制存储 I/O 操作**
- 传统方案（即使 GPUDirect Storage）CPU 始终占据控制路径 → 成为瓶颈
- SCADA 中：**GPU 担任"存储处理器"角色** — 发起千万级小存储请求，通过 ConnectX-9 传递给计算服务器
- 专门针对：**向量搜索、RAG、图分析、KV-cache 检索** 等细粒度随机读场景（<4KB 数据块，极端并行）
- Wiwynn 是首批展示 SCADA 服务器的 OEM 之一

**战略意义**:
- SCADA 属于 NVIDIA **Storage Next** 愿景 — 让存储行为接近 GPU 内存扩展
- 在 **PD 分离推理** 架构中，KV-cache 频繁访问存储的场景下，SCADA 的 GPU 直通存储路径可能带来数量级的性能提升
- PCIe 6.0 (Switch + SSD) 首次在 AI 存储服务器中进入实际应用

## 3️⃣ **Intel 计划 2027 H1 推出 "Raptor Lake Next" DDR4 平台 — 内存短缺驱动的倒退** 🔥 **NEW**

- **来源**: Tom's Hardware (Jake Roach, Jun 13, 2026)
- **来源链接**: https://www.tomshardware.com/pc-components/cpus/intel-reportedly-preparing-surprise-return-to-ddr4-systems-with-raptor-lake-next-ddr4-platform-slated-for-the-first-half-of-2027-on-the-lga-1700-socket-takes-a-page-from-amds-book-by-extending-budget-platform-longevity

**技术详情**:
- **产品**："Raptor Lake Next" — Raptor Lake 系列的第三次刷新（继 13/14 代之后）
- **时间**：**2027 年 H1**
- **插槽**：**LGA 1700**（维持与前代物理兼容性）
- **定位**：与 **Nova Lake**（预计 CES 2027 发布）并行销售，覆盖低端/预算市场
- **驱动因素**：**DDR5/DDR4 价差** — 内存短缺导致 DDR5 价格高企，Intel 借重返 DDR4 提供低成本选择
- **核心数量**：具体规格未知

**行业背景验证**:
- 至少 **2 家主板厂商** 确认向 Tom's Hardware 表示正在增加 **DDR4 主板产量**（同时包括 AM4 和 LGA 1700）
- AMD 最近重新推出 **Ryzen 7 5800X3D**（DDR4 平台），趋势一致
- Intel 的 Robert Hallock 此前表示 Raptor Lake 将保持"abundant availability"
- Intel 已为嵌入式/工业应用推出 **Bartlett Lake**（LGA 1700，纯 P-core，最高 12C），但消费者版本尚未明确

**战略意义 — 数据中心视角**:
- 虽然本文面向桌面/工作站市场，但反映了 **更广泛的 DDR4 供不应求趋势**
- AI 服务器大量消耗 DDR5 → 挤压消费级/工作站 DDR5 供应 → 推动 PC 市场 **退回 DDR4**
- 在服务器市场，DDR4 仍将长期与 DDR5 共存（低端/边缘场景）
- 历史对照：三星宣布 2026 年 DRAM 资本支出创历史新高（HBM + DDR5 优先）→ DDR4 产能受限

- **来源**: ServeTheHome (Vic A, Jun 2, 2026)

- **已在 `2026-06-13.md` 中完整覆盖**

**新增交叉视角**:
- Diamond Rapids (18A-P) 不依赖 TSMC → **Intel 唯一不受 TSMC 产能挤压的高端 CPU 路线**
- Google 已预订 Intel 封装 300 万+ TPU（2028）→ Intel 先进封装/制造的第三方业务正在加速
- Samsung 2nm 反攻延迟 → TSMC N2 成为高端芯片唯一选择 → **Intel 18A 的差异化窗口扩大**
