## 1️⃣ **NVIDIA 计划向中国销售 Vera CPU — GPU 封锁下的变通战略** 🔥

- **来源**: Tom's Hardware (Luke James, Jun 12, 2026)
- **来源链接**: https://www.tomshardware.com/pc-components/cpus/nvidia-preps-to-sell-its-vera-cpus-into-china-as-its-gpu-sales-stay-frozen

**技术详情**:
- NVIDIA 已告知中国客户，其 Arm 架构的 **Vera 服务器 CPU** 最早可在 **2026年8月** 对中国出货
- Vera CPU 基于 NVIDIA **Olympus CPU 核心**，88 核设计，TSMC **3nm** 工艺
- 当前 NVIDIA **GPU 对华销售全面冻结**（受美国出口管制限制），但 Vera CPU 可能不在管制清单内
- 若成功，这将成为 NVIDIA 在中国 AI 服务器市场的 **CPU-only 切入点**

**战略意义**:
- 美国出口管制迫使 NVIDIA 将 AI 战略拆分为「GPU 受限 / CPU 可售」两个通道
- Vera CPU 本身定位为 AI 集群的控制平面处理器 → 中国客户可通过购买 Vera CPU 保持与 NVIDIA 生态的兼容性
- 可能引发 **AMD EPYC Venice (Zen 6)** 在中国市场与 Vera CPU 的直接竞争

- 2026 年的竞争格局：**Venice (Zen 6) vs Vera (Olympus) vs Diamond Rapids (Xeon 7)** 三线并行

## 3️⃣ **Intel Xeon 7 Diamond Rapids 确认 2027 年发布：放弃 HT、取消 8 通道版本**

- **来源**: ServeTheHome (Vic A, Jun 2, 2026)
- **来源链接**: https://www.servethehome.com/intel-xeon-7-diamond-rapids-now-slated-for-2027/

**技术详情**:
- **工艺**: Intel **18A-P**（18A 的增强版本）
- **内存**: **16 通道** DDR5（带宽翻倍于 Xeon 6）
- **互联**: **PCIe Gen6** 支持
- **核心数**: 比 Xeon 6 **多约 50%**
- **HT（超线程）**: **不支持** — Intel 表示 HT 将在下一代 **Coral Rapids** 回归
- 取消了之前路线图中的 **8 通道版本**（此前由 STH Substack 独家披露）
- 采用 **可扩展 SoC 架构**，统一内存延迟

**竞争格局分析**:
- Diamond Rapids 在 **2027** 年上市时面临的市场：AMD Venice (Zen 6, 2026)、NVIDIA Vera (2026)、Arm AGI CPU、Qualcomm 传闻中数据中心 CPU、超大规模厂商自研芯片
- Intel 的最大优势：**使用 Intel Foundry 18A** 生产 → 不受 TSMC 产能挤压（AMD/Vera 都依靠 TSMC）

**交叉参考**:
- `knowledge/server-hardware/2026-06-10.md` 已有 Diamond Rapids 规格讨论 → 此处含完整细节和 HT 放弃的分析

- Cadence 同日宣布深化 Intel Foundry 合作（DIGITIMES, Jun 12）→ EDA 工具链层面为 Intel 18A/14A 铺路

## 5️⃣ **Gigabyte 40 节点 / 1U 超密度集群 — 新型服务器形态浮现**

- **来源**: ServeTheHome (Cliff Robinson, Jun 6, 2026)
- **来源链接**: https://www.servethehome.com/a-40-node-1u-cluster-gigabyte-r1c7-k0a-as1/

**技术详情**:
- **40 节点** Intel Core Ultra 7 258V (Lunar Lake)，每节点 4P+4E = 8 核
- **总计**: 320 核 + **1.28TB** LPDDR5X + **80 个** M.2 SSD + **40 个** iGPU (Intel Arc 集成)
- 外形: **1U** 机架式，5 个 8 节点弹匣式插槽（前 3 后 2）
- 电源: 2× **3.2kW Titanium** 级 PSU
- 网络: 后方 **2× QSFP28**（100GbE），内部通过交换芯片汇聚
- 管理: 机箱级 BMC，独立管理端口

**扩展数据**（STH 编辑推算）:
- **全机架 (40U)** 配置：12,800 CPU 核 + 3,200 SSD + 1,600 iGPU + 51.2TB LPDDR5X
- 仅需 80 个电源连接 + 80 个 100GbE 端口 + 40 个管理端口
- 密度高于 Arm AGI CPU 方案

**战略意义**:
- 新形态因子：将 GPU 卡的"弹匣"设计理念应用于 CPU 节点 → 面向 **CDN、视频转码、物理桌面虚拟化、边缘推理** 场景
- 无 ECC（Lunar Lake 片内封装 LPDDR5X 不支持 ECC）→ 不适合关键任务数据库，但适合水平扩展工作负载
- 与 Gigabyte 在 AI 服务器领域的布局一致（此前有 9U 8×B300 等产品）

**交叉参考**:
- `knowledge/server-hardware/2026-06-07.md` 已有 40 节点集群初步记录 → 此处补充完整规格和扩展数据
