## ④ Intel Xeon 7 Diamond Rapids 推迟至 2027

| 项目 | 内容 |
|:-----|:------|
| **来源** | [ServeTheHome - Intel Xeon 7 Diamond Rapids Now Slated for 2027](https://www.servethehome.com/intel-xeon-7-diamond-rapids-now-slated-for-2027/) |
| **发布方** | Intel（COMPUTEX 2026 路线图） |
| **工艺** | Intel **18A-P**（18A 增强版） |
| **预计上市** | **2027年** |

**规格升级**:
- **16 通道 DDR5**（内存带宽翻倍）
- **PCIe Gen 6** 支持
- 核心数比 Xeon 6（Granite Rapids）**增加约 50%**
- 统一内存延迟的可扩展 SoC 架构
- **不支持超线程**（Hyper-Threading 预计在下一代 Coral Rapids 回归）

**取消的SKU**: Intel 从路线图中移除了 8 通道 Diamond Rapids 版本，仅保留高端 SKU。

**竞争态势**: 2026 年 Intel 将面对 AMD EPYC、NVIDIA Vera（附带数百万 CPU 的 AI 集群）、Arm AGI CPU、高通传闻的新品以及超大规模厂商自研设计的多方竞争。

## ⑥ Intel + Foxconn Rackscale 战略合作

| 项目 | 内容 |
|:-----|:------|
| **来源** | [ServeTheHome - Intel Computex 2026 Keynote](https://www.servethehome.com/intel-computex-2026-keynote-live-coverage/) |
| **合作方** | Intel × **Foxconn** |
| **内容** | 联合开发 **Rackscale Blueprints**（机架级基础设施方案） |

**战略意义**: Intel 对标 NVIDIA/AMD 的机架级方案。Intel 强调不采用 one-size-fits-all，而是通过 OEM/ODM 合作伙伴提供定制化方案。

**SambaNova 联合展示**: SambaNova SN-50 Rack（Intel Xeons + SN-50 NPU），采用**解耦推理架构**（Disaggregated Inference）: GPU 做 Prefill + Prompt Caching，RDU 做 Decoding + Token Generation，CPU 做编排 + 工具执行。解耦方案比纯 GPU 方案快 **2-3 倍**。

## ⑦ Marvell T100 Teralink 交换机 + CPO 光互联突破

| 项目 | 内容 |
|:-----|:------|
| **来源** | [ServeTheHome - Marvell Computex 2026 Keynote](https://www.servethehome.com/marvell-computex-2026-keynote-live-coverage/) |
| **发布方** | Marvell（CEO Matt Murphy）|

**核心发布**:
1. **T100 Teralink 交换机** — 100T 容量以太网交换机
2. **CPO（Co-Packaged Optics）交换机** — 共封装光学交换机原型（体积远小于传统光模块交换机）
3. **COLORZ 1600** — 第4代硅光技术，用于跨数据中心长距离互联（数百到数千公里）

**Marvell 重大战略布局**:
- **收购 Celestial AI**（硅光子公司）
- **收购 XConn**（Scale-Up 交换芯片公司）
- 累计投资 **$360 亿** 建设数据中心基础设施平台
- 数据中心营收已占 Marvell 总营收 **75%**

**NVLink Fusion 合作**:
- NVIDIA 向 Marvell 投资 **$20 亿**
- Jensen Huang 亲临 Marvell 主题演讲站台
- NVLink Fusion 使 CSP 可将定制芯片与 NVIDIA 硬件搭配，构建异构数据中心

**铜→光互联拐点预测**:
- Marvell 明确提出「**200G per lane 将是最后一代铜缆能满足的速率**」
- 下一代将全面转向光互联
- 光互联可使 Scale-Up 域从 144 XPU 扩展到**数千 XPU**
- 未来可实现的完全解耦服务器架构（CPU、内存、XPU 分开放置）

## ⑧ 综合趋势判断

1. **NVIDIA 服务器平台化加速**: Vera Rubin 量产 + DGX Station Windows 化，NVIDIA 从卖 GPU 向卖完整服务器/工作站平台转型
2. **Intel 反击**: 18A 工艺 Clearwater Forest 288核出货 + Crescent Island 低成本推理卡 + Foxconn 机架级合作，但 Diamond Rapids 推迟到 2027 年暴露了 P-core 路线图断层
3. **光互联成为下一波主线**: Marvell + NVIDIA 联合推动 CPO 光互联，200G/lane 铜缆能力见顶，Scale-Up 域将从 144→数千 XPU
4. **Agentic AI 驱动 CPU 需求重回增长**: Intel 强调 Agent 推理需要大量 CPU 编排，不再是纯 GPU 工作负载
5. **解耦推理架构从实验走向部署**: Intel + SambaNova 展示 Prefill/Decode/编排三阶段分离，性能 2-3x 优于纯 GPU 方案
