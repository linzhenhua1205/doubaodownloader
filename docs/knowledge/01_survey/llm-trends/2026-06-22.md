## 🚀 模型与产品动态

### ④ Google DeepMind 发布 AI Control Roadmap（Agent 安全框架）🔒
- **时间**: 6/19
- **核心内容**:
  - 类比"像带双刹的驾校教练"——信任 Agent 但随时准备好接管或刹车
  - 五大安全机制:
    - **Chain-of-Thought 监控**: 实时追踪模型推理过程
    - **异步告警 (Asynchronous Alerts)**: Agent行为异常时发出警报
    - **实时访问控制 (Real-time Access Control)**: 动态调整Agent权限
    - **关停基础设施 (Shutdown Infrastructure)**: 在必要时可立即停止Agent运行
    - **内部护栏 (Internal Guardrails)**: 应对越来越难以监管和控制的对抗行为
  - **意义**: 最早的主力 AI 厂商之一发布系统级 Agent 安全框架

## 📄 arXiv 论文精选（后训练/RL/对齐）

### ⑦ VIMPO: Value-Implicit Policy Optimization for LLMs 🔬
- **链接**: [arXiv:2606.20008](https://arxiv.org/abs/2606.20008)
- **作者**: Zhewei Kang, Aosong Feng, Sergey Levine, Dawn Song, Xuandong Zhao (UC Berkeley)
- **核心**: 无需 Critic 的 RLVR 方法
  - 从 **KL-正则化 RL 的最优条件** 推导出 **policy 隐式价值函数**
  - 自回归生成时，价值递归可写作 policy-reference log 比率，由轨迹终点零未来奖励条件锚定
  - 价值损失融入结果级可验证奖励，无需训练 critic
  - 同源推导也给出无 critic 的 actor advantage
- **结果**: MATH-500 / AIME 2024 / AIME 2025 / OlympiadBench 全面超越 **GRPO**，尤其在竞赛级推理和噪声奖励场景优势显著
- **意义**: 结合了 GRPO 的简洁（无 critic）和 actor-critic 的细粒度信用分配

### ⑧ Connect the Dots (CoD): 用 RL 训练 LLM 长期 Agent 跨域泛化 🔬
- **链接**: [arXiv:2606.20002](https://arxiv.org/abs/2606.20002)
- **作者**: Yanxi Chen 等 (阿里巴巴)
- **核心**: 训练 LLM 具备"长期 Agent 元能力"
  - LLM Agent 在环境中部署后，解决长序列任务，同时持续探索、从自身经验学习、迭代更新上下文
  - 包含: (1) GRPO-style RL 算法 + 细粒度信用分配; (2) 激励和衡量该元能力的任务和环境
- **结果**: 端到端 RL 训练在该设定下有效，具备 **域内/跨域/到 Ralph-loop 设置的 OOD 泛化** 能力
- **意义**: 首次系统性展示 RL 可以训练 LLM 获得"长期经验复用"的元能力

### ⑨ Sequential DPO 非均匀遗忘研究 🔬
- **链接**: [arXiv:2606.19744](https://arxiv.org/abs/2606.19744) (已提交 EMNLP 2026)
- **作者**: Pranav Bhandari 等
- **核心**: 顺序 DPO 不会导致统一的遗忘模式
  - 偏好变化范围从部分退化到稳定、pair-level 重分布、或正向迁移
  - 取决于目标关系、信号强度、训练顺序
  - 长归一化 policy margin 的 **四分位数分解** 揭示了 aggregate 指标掩盖的异质性
  - Stage 2 梯度与前一目标的适配器更新近乎正交，直接梯度对抗不是主因
- **意义**: 未来顺序对齐管线应评估目标兼容性和信号强度，而非假设"后训练统一覆盖前训练"
