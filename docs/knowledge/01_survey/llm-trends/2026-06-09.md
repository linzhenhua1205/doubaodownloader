### 2️⃣ "Chat is dead" — OpenAI 最大规模重构：ChatGPT 变身超级应用

**来源**: Financial Times / Ars Technica (Jun 8)
**链接**: [Ars Technica](https://arstechnica.com/ai/2026/06/chat-is-dead-openai-preps-overhaul-of-chatgpt/)

OpenAI 高级员工坦言 **"Chat is dead"**，ChatGPT 将迎来"史上最大改版"：

- **超级应用战略**：ChatGPT 将在"未来几周"开始变身，整合 coding 工具（Codex）、AI Agent、图像生成、第三方伙伴应用（Canva、Booking.com 等）于一体
- **Codex 6 倍增长**：Codex 的周活用户从 2 月桌面应用发布以来已增长 **6 倍 → 500 万+ WAU**
- **收入结构天平倾斜**：200 万企业客户贡献约 40% 收入，目标年底达 50%
- **核心逻辑**：OpenAI 认定 AI 的未来是 **Agent 而非 Chatbot**。ChatGPT 是"入口"，Codex / Agent 是"高毛利产品"
- **界面重构**：新 prompts/features 引导用户使用 coding、图像生成、Agent 任务；后续将让模型自动理解用户意图
- **已关闭**：Sora（视频生成产品上线不到一年关闭）、ChatGPT 内购功能
- 与 Anthropic 策略收敛：此前 Anthropic 侧重企业赚钱，OpenAI 侧重"大棋"，现在双方都在冲刺 IPO，投资者更关心盈利能力

### 3️⃣ 🚨 Apple AI 跑在 Nvidia 硬件上 — WWDC 2026 最大意外

**来源**: The Verge (Jun 8), Ars Technica (Jun 8), TechCrunch (Jun 8-9)
**链接**: [The Verge WWDC live blog](https://www.theverge.com/2026/6/8/24968393/apple-ai-runs-on-nvidia-chips-wwdc) | [Ars Technica Siri AI 详解](https://arstechnica.com/ai/2026/06/say-hi-to-siri-ai-apple-announces-new-more-conversational-voice-assistant/)

WWDC 2026 技术分享会上，Apple 透露了一个震撼行业的消息：**Apple Foundation Model 跑在 Nvidia 硬件上、部署在 Google Cloud 中**。

**Siri AI 核心架构**：
- **双轨制 AI 模型**：高端设备（iPhone Air/17 Pro、M4+/12GB+）获得"最强大模型"；低端设备获得阉割版（缺少表达性语音、低精度听写）
- **Google Gemini 基础模型**：基于 Apple 2025 年与 Google 签订的多年代理协议
- **云侧跑在 Nvidia 硬件上**：Private Cloud Compute 使用 Nvidia GPU（与 Google/Intel 合作部署）
- 新功能：屏幕理解、多步 Agent 任务、跨 App 上下文、Siri 独立 App（跨设备对话漫游）
- 秋季发布，仅英语先行

> 意义：**Apple（长期反 Nvidia、自研 AI 芯片的公司）最终仍不得不求助于 Nvidia 硬件**，标志着 Nvidia 在 AI 基础设施中的统治地位进一步巩固。

### 4️⃣ Google 转单 Intel 制造 TPU — 3M+ 芯片改写代工格局

**来源**: The Verge (Jun 8, 引用 The Information)
**链接**: [The Verge](https://www.theverge.com/2026/6/8/24968936/google-intel-manufacture-ai-chips)

受 TSMC 产能限制，Google 将转向 **Intel Foundry 制造 TPU**：
- Intel 将在 2028 年代工 **300 万+ 块 TPU**（Google 预计 2027-2028 共生产 ~600 万块 TPU，一半交给 Intel）
- Nvidia 和 SK Hynix 也正在 **测试 Intel 代工技术**
- 这是 Intel Foundry 在 AI 代工领域的最大突破。结合此前 COMPUTEX 2026 上 Intel Xeon 6+（288 核 18A）和 Crescent Island 推理芯片，Intel 正从数据中心 CPU 向 AI 全栈代工转型

### 5️⃣ 🚨 npm 凭证窃取攻击精准瞄准 AI Agent — 73 个 Microsoft 签名包被投毒

**来源**: Ars Technica (Jun 8)
**链接**: [Ars Technica](https://arstechnica.com/security/2026/06/for-the-2nd-time-in-weeks-microsoft-packages-laced-with-credential-stealer/)

**严重安全事件** - Miasma 蠕虫攻击，这是两个月内第二次突破官方 Microsoft 仓库账号：

- **73 个经 GitHub 加密验证的 Microsoft npm 包**被植入凭证窃取代码（攻击者盗用 Microsoft 正式发布凭证）
- **触发条件**：当开发者用 AI Agent（Claude Code、Gemini CLI、Cursor、VS Code）**打开这些包时自动触发**
- **Miasma 蠕虫**（28KB 载荷）：窃取 AWS/Azure/GCP/K8s/密码管理器/90+ 开发者工具凭据，通过云基础设施横向扩散
- **突破 SLSA 验证**：攻击利用了 SLSA 证明的信任模型——用盗窃的合法凭证生成有效 OIDC token，签名通过了供应链验证
- **每个感染生成唯一加密载荷**，传统基于哈希的检测完全失效
- **同一个 Microsoft 账号在 5 月已被攻破一次**（durabletask Python SDK），此次是"二次攻破"

### 7️⃣ arXiv 论文亮点（6月8-9日提交）

**今日 244 篇新论文**（cs.CL），选录与 LLM/Agent 直接相关的亮点：

- **IS-CoT** (2606.09709)：交错式结构化思维，解决长文本生成崩溃问题
- **End-to-End Context Compression at Scale** (2606.09659)：大规模端到端上下文压缩
- **EntropyInfer** (2606.09508)：基于熵的自适应长上下文推理，Llama/Qwen/openPangu 系列上 2.39×加速（100K+ tokens）
- **Emergence World** (2606.08367)：跨厂商多 Agent 15 天长期仿真平台（Claude Sonnet 4.6 vs Grok 4.1 Fast vs Gemini 3 Flash vs GPT-5-mini vs 混合群），发现同类初始条件下的行为**彻底分化**——从稳定的审议式治理到全种群崩溃
- **LATTEArena** (2606.09004)：首个 LLM 表格特征工程标准化评测框架，15 种方法六维分解消融
- **Hacker-Fixer Loop** (2606.08960)：自动化 Agent 基准加固，审计发现 **16% 任务可被黑客攻击**（1,968 任务中 323 个），用 3 Agent 循环修复至 0%
- **AEDI 谄媚指数** (2606.07897)：首个连续指标评估模型对用户态度敏感度，Claude 最抗谄媚，Grok/Gemini 最严重

3. **Nvidia 不可撼动**：连 Apple 都不得不回归 Nvidia 硬件，GPU 短缺将延长 Nvidia 统治期
