### 1. Anthropic Claude Sonnet 5 正式发布——Agent 化能力逼近 Opus 级别

**来源**: [Anthropic Blog](https://www.anthropic.com/news/claude-sonnet-5) · 2026-06-30 · [The Verge 报道](https://www.theverge.com/ai-artificial-intelligence)

**定位**: 迄今最具 Agent 能力的 Sonnet 系列模型，Sonnet 4.6 的后继者

| 维度 | 数据 |
|:-----|:------|
| **能力对标** | **性能接近 Opus 4.8**，但不是直接替代（保留 Opus 4.8 的独立定位） |
| **Agent 能力** | 可自主制定计划、使用浏览器/终端、运行自主工作流——数月前仅 Opus 级模型可用 |
| **定价** | 发布特惠 $2/M 输入 + $10/M 输出（至 2026-08-31），之后 $3/$15 |
| **对比 Opus 4.8** | Opus 4.8 定价 $5/$25，故 Sonnet 5 是 **40%-60% 价格获相近能力** |
| **Humanity's Last Exam** | 34.6%（无工具）/ 46.8%（带工具） |
| **OSWorld-Verified** | 78.5% |

**能力深度**:
- **Agentic 搜索 (BrowseComp)**: Sonnet 5 覆盖更广的 cost-performance 曲线，高 effort 模式下可匹配 Opus 4.8
- **Computer Use (OSWorld)**: 同样在中等 effort 显著优于 Sonnet 4.6
- 推理/工具使用/编码/知识工作**全面改善**

**安全评估**:
- 不良行为整体**低于** Sonnet 4.6（自动化行为审计更低）
- **拒绝攻击/即时注入抵抗**能力提升
- 幻觉/谄媚率降低
- **网络安全能力远低于** Opus 4.8 / Mythos 5：Firefox 漏洞利用评估中 **0% 完全成功率**（vs Opus 4.8 和 Mythos 5 有显著能力）
- 默认启用网络安全 safeguard（与 Opus 4.7/4.8 同级）

**行业意义**: Sonnet 5 实际上缩小了"中档模型"与"旗舰模型"之间的 Agent 能力鸿沟，标志着 Agent 能力正在从旗舰级下沉到主流定价区间。

### 2. Anthropic Fable 5 全球恢复部署——美国政府解除出口管制

**来源**: [Anthropic Blog - Redeploying Fable 5](https://www.anthropic.com/news/redeploying-fable-5) · 2026-06-30 · 生效 2026-07-01

**事件回顾**:
| 日期 | 事件 |
|:-----|:------|
| Jun 9 | Fable 5 和 Mythos 5 发布 |
| Jun 12 | 美国政府因 Amazon 报告发现 Fable 5 安全 bypass 而对两模型施加出口管制 |
| Jun 26 | Mythos 5 获准恢复对 100+ 美国组织/公司的有限访问 |
| Jun 30 | **两模型出口管制完全解除** |
| Jul 1 | Fable 5 全球恢复，Mythos 5 同步扩大访问 |

**关键事实核查**: Amazon 报告中 Fable 5 的 bypass 安全问题，经 Anthropic 验证：
- 其他模型（Opus 4.8, GPT-5.5, Kimi K2.7, 甚至 Haiku 4.5）**同样能识别相同漏洞**
- 漏洞利用演示代码**所有测试模型都能生成**
- 报告技术**未暴露 Mythos 级独特能力**，属于安全边际内的边界情况

**行业标准倡议**: Anthropic 联合 Amazon、Microsoft、Google 等 Glasswing 合作伙伴，**发起行业首个 AI Jailbreak 严重性评分框架**，四维评分：
1. **能力增益**: 越狱解锁的能力超出已有工具的幅度
2. **能力广度**: 同一种越狱能触及多少种不同攻击任务
3. **武器化难易**: 从越狱到实际攻击需要多少人力
4. **可发现性**: 越狱技术的传播广度

**政府协作升级**:
- 前沿模型**预发布政府访问与评估**
- **快速信息共享**机制：安全漏洞/越狱即时通知
- **联合研究资源**: Anthropic 设立专门团队配合政府安全研究
- **行业通用标准**: 推动建立自愿性的前沿模型安全评价标准

**影响判断**: 这是 AI 监管历史上的标志性事件——政府直接干预模型发布，随后又经谈判解除。未来 <100B 级安全审查可能成为常态。

### 🧠 GPT-5 在 Agent 心理理论（ToM）任务中首次超越人类

**论文**: [NCP-ExploreToM: Theory of Mind and Persuasion Beyond Conversation](https://arxiv.org/abs/2606.31916)
**来源**: arXiv:2606.31916 · Cambridge 等

| 维度 | 数据 |
|:-----|:------|
| **任务** | Non-Conversational Planning ToM (NCP-ToM)：通过行动诱导其他 Agent 的信念状态 |
| **测试范围** | 6 个前沿模型（GPT-5, Gemini 2.5 Pro, Claude 4 系列）+ 人类对照，600 个实例 |
| **GPT-5 表现** | 约 **80% 成功**，**唯一在所有场景下超越人类的模型** |
| **其他发现** | 所有模型（像人类一样）在诱导真实信念 vs 虚假信念时表现更好 |
| **意义** | 非对话场景下的心理理论能力对自主 Agent 安全有重要意义 |

### ⚡ CoLT: 潜在思维链——推理加速 10.1×，无需语言中间步骤

**论文**: [CoLT: Teaching Multi-Modal Models to Think with Chain of Latent Thoughts](https://arxiv.org/abs/2606.31986)
**来源**: arXiv:2606.31986 · ECCV 2026

| 维度 | 数据 |
|:-----|:------|
| **方法** | 用潜在表示取代冗长文本中间推理步骤，最少仅需 3 步 |
| **加速** | 推理时间减少 **10.1×**，文本解码时间减少 **22.6×** |
| **监督** | 外部轻量 Decoder 提供前向+后向逐步监督，推理时去除 |
| **结果** | 8 个基准上超越现有潜在推理方法（CODI, SIM-CoT），持平或超越文本 CoT |

**意义**: 去除了 CoT 推理的文本瓶颈，为多模态推理的实时部署打开前景。

### 🏥 MRPO: RL 医学推理——早期推理失败从 64% 降至 13%

**论文**: [Breaking Failure Cascades: Step-Aware RL for Medical Multimodal Reasoning](https://arxiv.org/abs/2606.31825)
**来源**: arXiv:2606.31825 · Korea University

| 维度 | 数据 |
|:-----|:------|
| **方法** | 步骤级别过程奖励：最终答案错误时对早期无效推理步骤指数级惩罚 |

| **效果** | 超越更大的 HuatuoGPT-Vision-34B 达 **2.79 分** |
| **失败率** | 早期推理失败从 **64.0% → 13.0%** |
| **适用性** | 3 个 MLLM 骨干上一致优于标准 GRPO |

### 🔪 ERA: MLLM 视觉 Token 剪枝——降低 33.7% TFLOPs，保留 99.5% 性能

**论文**: [ERA: Entropy-Guided Visual Token Pruning with Rectified Attention](https://arxiv.org/abs/2606.31982)
**来源**: arXiv:2606.31982

- 提出 "Attention Logit Collapse" 现象：现有 token 剪枝方法扭曲注意力分布
- 三组件方案：DEP（熵引导剪枝）+ BTR（回收）+ LAR（注意力纠正）
- 3 种 MLLM 架构 + 10 个 VQA 基准验证，代码已开源

### 🔬 GR2: LLM 重排序——工业推荐系统 +18.7% R@1

**论文**: [GR2 Technical Report](https://arxiv.org/abs/2606.31984)
**来源**: arXiv:2606.31984

- 端到端 LLM 框架用于推荐系统重排序环节
