## 2️⃣ 🟣 Anthropic Claude Opus 4.8 发布

| 维度 | 详情 |
|:-----|:------|
| **来源** | [Anthropic 官方博文](https://www.anthropic.com/news/claude-opus-4-8)，2026-05-28 |
| **定价** | 输入 $5/M tokens，输出 $25/M tokens（与 Opus 4.7 **同价**）|
| **Fast Mode** | 2.5× 速度，$10/$50 per M tokens，比前代便宜 **3 倍** |

### 📊 核心评测数据

| 基准测试 | Opus 4.8 | Opus 4.7 | 对比 |
|:---------|:---------|:---------|:-----|
| **SWE-bench Verified** | **72.8%** | 71.5% | +1.3pp |
| **Terminal-Bench 2.1** | **48.4%** | 46.1% | +2.3pp |
| **OSWorld-Verified** | **84.1%** | 82.3% | +1.8pp |
| **TAU-bench (retail)** | **72.0%** | 71.5% | +0.5pp |
| **TAU-bench (airline)** | **82.6%** | 82.3% | +0.3pp |
| **Online-Mind2Web** | **84%** | 显著优于前代 | 最强浏览器Agent |
| **Legal Agent Benchmark** | **>10% all-pass** | 未达10% | 首个突破 |
| **Super-Agent Benchmark** | **100%用例完成** | 未达 | 唯一端到端全通过 |

### 🎯 关键能力突破
- **诚实度大幅提升** — 约 **4 倍**于 Opus 4.7 更少隐瞒代码缺陷，主动标记不确定性
- **Alignment 评估** — 亲社会特质达新高（用户自主性支持、用户利益优先），失调行为（欺骗/滥用合作）显著低于 Opus 4.7，接近 **Claude Mythos Preview** 水平
- **Dynamic Workflows** — Claude Code 可规划并运行 **数百个并行子Agent**，已验证完成数十万行代码的全仓库迁移
- **Effort Control** — 用户可调节 Claude 的思考力度（低/高/额外/最大），"Max" 模式用于困难任务和长时间异步工作流

### 🏢 合作伙伴评价
- **Cursor**：在所有 effort 级别上超越前代，工具调用更高效
- **Devin**：解决了 Opus 4.7 的注释冗长和工具调用问题
- **Databricks Genie**：解锁 Agent 推理阶跃提升，多模态可推理 PDF/图表，token 成本比 Opus 4.7 便宜 **61%**
- **Hebbia**：引文精度更好，token 效率更高

### 🔮 未来路线
- **Mythos Preview** — 目前仅 Project Glasswing 组织使用，强化安全防护后将在 **数周内** 向所有客户开放（高于 Opus 的能力等级）
- 计划发布 **更低成本版本** 提供 Opus 级能力
