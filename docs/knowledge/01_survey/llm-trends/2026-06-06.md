## 📄 arXiv 重要论文（6月4-5日提交）

### 🔬 CLSA — 微软跨层稀疏注意力
**论文**: [You Only Index Once: Cross-Layer Sparse Attention with Shared Routing](https://arxiv.org/abs/2606.06467)
**作者**: 孙宇涛, 张岩琦, 董力, 王建勇, 魏福来
- 核心创新：基于 YOCO KV-sharing 架构，跨解码器层共享 **路由索引**
- 一个 indexer 一次性计算 token-level top-k 选择，所有层复用
- **性能**：128K 上下文下 **7.6× 解码加速**，**17.1× 整体吞吐提升**
- 同时优化 prefilling、KV-cache 存储、长上下文解码三大瓶颈

### 🔬 Vortex — 可编程稀疏注意力服务系统
**论文**: [Vortex: Efficient and Programmable Sparse Attention Serving for AI Agents](https://arxiv.org/abs/2606.06453)
**作者**: Zhuoming Chen 等 (CMU)
- Python 嵌入前端 + page-centric tensor 抽象，支持广泛的稀疏注意力算法
- AI Agent 自动生成和迭代多样化算法，**最高 3.46× 吞吐优于 full attention**
- **扩展到 MLA 架构**：GLM-4.7-Flash 上达 **4.7× 更高吞吐**
- MiniMax-M2.7 (229B) 上达 **1.37×** 吞吐提升（NVIDIA B200 GPU）

### 🔬 YouZhi — 邮储银行 × 华为金融 LLM
**论文**: [YouZhi: Towards High-Concurrency Financial LLMs via Adaptive GQA-to-MLA Transition](https://arxiv.org/abs/2606.05868)
- 层自适应 **GQA→MLA** 过渡框架，动态分配每层 FreqFold 大小
- **在华为昇腾 NPU 上运行**（vLLM-Ascend）
- YouZhi-7B：金融评测 +12.3%，最大并发 **2.69×**
- YouZhi-14B：金融评测 +7.0%，最大并发 **2.43×**

### 🔬 ReasoningFlow — 推理结构分析
**论文**: [ReasoningFlow: Discourse Structures for Understanding LLM Reasoning Traces](https://arxiv.org/abs/2606.05402)
- 对 5 个模型（Qwen2.5-32B、QwQ-32B、DeepSeek-V3、DeepSeek-R1、**GPT-oss-120B**）的 1,260 条推理轨迹进行自动标注
- **核心发现**：不同模型推理结构**惊人相似**（即便不同基座和训练数据）
- 大多数错误步骤未用于推导最终答案
