# ☸️ 云原生

> Kubernetes、容器生态、CNCF项目、微服务架构、Serverless

## 范围
- **容器编排**: Kubernetes（K8s）核心机制、调度、网络、存储
- **容器运行时**: Docker、containerd、runc、cri-o、WASM
- **容器网络**: CNI、Calico、Cilium/eBPF、Flannel
- **容器存储**: CSI、Rook/Ceph、Longhorn
- **服务网格**: Istio、Linkerd、Envoy
- **CNCF生态**: Prometheus、ETCD、Harbor、Knative、KEDA 等毕业/孵化项目
- **微服务架构**: 12-Factor App、DDD、API Gateway、配置中心
- **Serverless**: Knative、OpenFunction、函数计算
- **CI/CD**: ArgoCD、Tekton、GitOps
- **多云/混合云**: Kubernetes Federation、Cluster API、服务网格跨集群
- **云原生安全**: Pod Security Standards、OPA/Gatekeeper、CVE管理、供应链安全
- **可观测性**: Prometheus/Grafana、OpenTelemetry、Loki、Tempo

## 标签
`Kubernetes` `容器` `CNCF` `微服务` `服务网格` `Serverless` `CI/CD` `云原生安全` `可观测性`
