## 1️⃣ GitHub Code Quality — 正式发布为付费产品，7月20日生效 🏗️

- **来源**: [GitHub Changelog (Jun 16, 2026)](https://github.blog/changelog/2026-06-16-github-code-quality-generally-available-july-20-2026/)
- **工具**: GitHub Code Quality
- **标签**: `github` `code-quality` `ga` `enterprise` `codeql` `paid-product`

**核心内容**:

GitHub 宣布其 **Code Quality** 产品将于 **2026年7月20日** 从公开预览转为付费 GA。超过 **10,000 家企业** 已在预览期间使用。

### 产品定位

Code Quality 是 GitHub 在代码质量管理领域的全新产品线，之前主要聚焦安全（Code Scanning/Secret Scanning/Dependabot），现在扩展到**可维护性、可靠性、代码覆盖率**三个质量维度。

### 定价结构（三部分计费）

| 组件 | 计价方式 | 覆盖范围 |
|:-----|:---------|:---------|
| **Base subscription** | **$10/活跃提交者/月**（按启用仓库计） | findings · scoring · rulesets 集成 · quality gates 阻止 PR 合并 · 组织级仪表盘 |
| **AI 能力** | **按用量计费**（usage-based） | Copilot code review + AI-assisted detection + Copilot Autofix 的 AI 消耗 |
| **确定性分析** | **GitHub Actions 分钟**（免费，不用 AI 计费） | CodeQL 驱动的 maintainability/reliability 扫描 |

### 功能矩阵

| 功能 | 说明 |
|:-----|:------|
| **组织级部署** | 一次性配置，覆盖全仓库 |
| **组织级质量仪表盘** | 跨仓库的可维护性/可靠性/覆盖率的统一视图 |
| **Quality Gates** | 基于 rulesets 在 PR 合并前自动阻止无法通过质量阈值的变更 |
| **组织级/仓库级质量评分** | 质量画像量化评分 |
| **API** | 启用管理 + findings 查询 API |

### 可用性

- **平台**: GitHub Enterprise Cloud + GitHub Team 计划
- **不支持**: GitHub Enterprise Server

### 🚨 重要操作提醒

> 如果不想在7月20日后继续使用 Code Quality，**请在 7月20日之前从组织设置中禁用**（今日起已可在 settings 中操作）。

## 2️⃣ Code Quality — 组织级一键启用/禁用 🎛️

- **来源**: [GitHub Changelog (Jun 16, 2026)](https://github.blog/changelog/2026-06-16-organization-level-enablement-for-github-code-quality/)
- **工具**: GitHub Code Quality
- **标签**: `github` `code-quality` `organization` `enterprise` `admin`

**核心内容**:

与 GA 公告同日发布——组织管理员现在可以在 **Settings → Security → Code quality** 中通过**一个开关**为组织中**所有仓库**启用或禁用 GitHub Code Quality，不再需要逐个仓库配置。

**可用性**:
- 今日起在 GitHub Enterprise Cloud + Team 计划的 **public preview** 中可用
- 不支持 GHES

**价值判断**: 对于大型组织（1,000+ 仓库），逐个仓库启用/禁用是不可能的。组织级开关是 Code Quality 从"可选工具"变为"组织级质量策略"的必要前提——与 rulesets 配合，可在**几小时内完成全组织的质量门禁部署**。

## 📈 趋势观察（2026-06-17）

**GitHub Code Quality 的产品意义**: 这是 GitHub 在 **"安全→质量"** 维度上的重大扩展。此前 GitHub Advanced Security 覆盖安全（Code Scanning + Secret Scanning + Dependabot），Code Quality 将质量门禁（可维护性/可靠性/覆盖率）提升到同等地位。与 CodeQL 的确定性分析 + Copilot 的 AI 审查形成双轮驱动。$10/committer/月的定价与 Advanced Security（$49/committer/月）相比低很多，更容易被团队接受。对于 CodeQL 团队，这是其确定性分析能力在"安全之外"的新应用场景。
