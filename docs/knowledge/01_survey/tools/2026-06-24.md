## 1️⃣ JetBrains Junie — AI Coding Agent 正式 GA 🏆

- **来源**: [JetBrains Blog (2026-06-22/23)](https://blog.jetbrains.com/junie/2026/06/junie-coding-agent-out-of-beta/)
- **工具**: JetBrains Junie
- **标签**: `jetbrains` `junie` `ai-agent` `ga` `coding-agent` `swe-rebench`

**核心内容**: Junie 经过一年实验和 Beta 测试，正式进入 General Availability。以下是 GA 版本四大核心能力：

### 🔹 Plan Mode — 先规划再编码

- 编码前生成结构化文档（Product Requirements / Technical Design / Delivery Stages / Testing Strategy）
- 文档是**真实文件**（`.junie/plans/`），可提交到版本控制
- 需求模糊时，Agent 主动追问（多选/自由回答），**不猜**
- 用户审批 plan 后 Junie 才开始实现
- **Shift+Tab** 进入 Plan Mode，**Ctrl+P** 打开 plan 文档
- 核心价值：Plan 用贵模型，实现用便宜模型，减少无效 token 消耗

### 🔹 Agentic Debugging — 用调试器而非 println

- Junie 可**驱动 IDE 调试器**：启动调试会话、设置断点（项目代码/库代码/SDK 代码/反编译 .class 文件/JAR 资源）
- 检查**真实运行时状态**：栈帧、线程状态、表达式求值、run-to-line
- 支持三种调试模式：
  - 完全自主：`"Debug and figure out why this test fails only on the second iteration."`
  - 人触发 Agent 设断点：`"Prepare the debugger, I'll trigger the UI flow."`
  - 人调试中委托检查：`"Continue my current debug session and tell me why this value becomes null."`

### 🔹 Remote Control — 异步任务 + 远程监控

- 从笔记本启动任务，手机查看进度，咖啡时间 review PR
- Junie 异步执行，任意设备登录即可查看 session 状态

### 🔹 Code Review — 全项目上下文

- 支持 GitHub Actions / GitLab（含 on-prem）/ CLI `/review` 命令
- 范围可选：unstaged changes / staged changes / diff against main
- **交互式 walkthrough**：逐文件高亮、解释设计决策、inline accept/reject + PR 评论
- 可根据关注的上下文**动态重排剩余 review 顺序**

### 🔹 SWE-Rebench 成绩

- SWE-Rebench 最新一轮（独立基准，每次轮换新题）Junie **排名第一**
  - **61.6% resolved**
  - **72.7% pass@5** — 超越其他 AI coding agent，与原始 frontier 模型竞争

### 🔹 架构特点

- **LLM-agnostic**：支持任何模型（Anthropic / OpenAI / Google / 本地运行时 via Ollama/LiteLLM/LMStudio）
- **ACP 协议**：同一引擎跨 AI Chat / Junie Tool Window / Junie CLI 三界面
- **Database 集成**：通过 DataGrip / Database Plugin 连接 IDE 已配置的数据库，查询真实数据、写 SQL
- **BYOK 支持**：自带密钥

## 3️⃣ VS Code 1.125 — 集成浏览器 + MDM Copilot 策略 🚀

- **来源**: [VS Code 1.125 Release Notes (Jun 17, 2026)](https://code.visualstudio.com/updates/v1_125)
- **工具**: Visual Studio Code
- **标签**: `vscode` `1.125` `integrated-browser` `mdm` `copilot` `lsp`

**核心内容**: 1.125 版本带来多个值得关注的新特性：

### 🔹 集成浏览器（正式版）
- 地址栏支持**Web 搜索**（可配置搜索引擎 `workbench.browser.searchEngine`）
- **远程连接代理**（Preview）：在 remote workspace 中，HTTP(S) 流量通过远程连接代理，安全访问仅远程可达的端口和服务
- Agent 操作转发端口时自动处理 URL 重写

### 🔹 Copilot 企业管控
- **原生 MDM 交付**：管理员可通过 Windows/macOS MDM 渠道推送 Copilot 策略设置，不依赖用户登录，自动生效且不可本地覆盖
- 扩展自动更新支持**延迟配置**：`extensions.autoUpdateDelay`（默认 2 小时，企业可管控）
- 扩展自动更新简化为 on/off，禁用扩展不再自动更新

### 🔹 模型提供商管理
- Language Models 编辑器新增 **Install Model Providers** 按钮，自动过滤可用的模型扩展
- 支持查看 Copilot **附加消费**使用百分比，帮助控制预算超支

### 🔹 LSP 3.18
- Language Server Protocol 升级至 3.18，对应 `vscode-languageclient@10.0.0` 和 `vscode-languageserver@10.0.0`
