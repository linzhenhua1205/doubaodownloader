## ① 🔥 **Supermicro GB300 Super AI Station 详细拆解——桌面级 GB300 超节点首度披露**

**时间**: 2026-06-27 · **来源**: ServeTheHome（Ryan Smith）
**原文标题**: 「Taking an Up-Close Look at the Supermicro GB300 Super AI Station」
**链接**: https://www.servethehome.com/taking-an-up-close-look-at-the-supermicro-gb300-super-ai-station/

### 概述

Supermicro 在 Computex 2026 上展示了其 **DGX Station 系统——Super AI Station**，这是首款公开详细拆解的 **桌面级 GB300 超节点**。NVIDIA 于 Computex 发布多款桌面系统设计后，Supermicro 成为首批拿到 GB300 桌面方案的 OEM 之一。

### 核心架构

| 维度 | 规格 | 说明 |
|:-----|:-----|:-----|
| **SoC** | Grace Blackwell **GB300** | 72-core Grace CPU + Blackwell Ultra GPU 整合封装 |
| **功耗 (TDP)** | **1,600W** | 需独立供电回路 |
| **冷却** | **全液冷** | 冷板覆盖所有关键组件 + 前置 radiator 散热排 |
| **GPU 内存** | **252GB HBM3e**（7/8 stacks enabled） | 7.1 TB/s 带宽（de-rated 版，非全 8-stack） |
| **系统内存** | **496GB LPDDR5X**（4× SOCAMM） | 为 Grace CPU 提供，冷板覆盖，不可轻易插拔 |
| **存储** | 4× **PCIe Gen5 x4 M.2 2280** | 展示用 480GB Micron 7450 Gen4 驱动 |
| **网络** | **2× 400GbE QSFP**（ConnectX-8） | NVIDIA ConnectX-8 双端口 400GbE |
| **扩展槽** | 3× PCIe x16（1 个已用 RTX PRO） | 1 个 RTX PRO 显卡位（图形输出）+ 2 个空闲 |
| **尺寸** | 桌面式，可配 **5U 机架安装套件** | 兼顾桌面开发与机架部署 |

### 液冷设计细节

```
  ┌─────────────────────────────────────────┐
  │  Rear (GB300 + motherboard)              │
  │  ┌───────────────────────┐              │
  │  │  GB300 SoC (冷板)     │              │
  │  │  LPDDR5X SOCAMM (冷板)│              │
  │  │  ConnectX-8 (散热片)  │              │
  │  └───────────────────────┘              │
  │                   ↓                      │
  │  ┌───────────────────────┐              │
  │  │  Radiator (前置散热排)  │ ← 3 fans    │
  │  └───────────────────────┘              │
  │  Front (通风面板 + USB)                  │
  └─────────────────────────────────────────┘
          气流: 前→后 (前置 radiator → 吹过系统)

冷板覆盖: GB300 SoC + 4× LPDDR5X SOCAMM
散热排: 3× 风扇，前置，拉风式
```

> 💡 **关键架构决策**: 前置 radiator + 前→后气流的配置是 **刻意为之**——这样桌面系统可无缝转换为 **5U 机架部署**。虽然不如 HGX 服务器空间效率高，但对需要**单 GB300 入柜**的客户提供了便利。

### 超节点意义

| 角度 | 分析 |
|:-----|:------|
| **定位** | 填补了 HGX 大服务器与工作站之间的空隙——**桌面级超节点**，$125K 价位 |
| **冷却** | 1.6kW 单系统液冷方案验证了 GB300 在 **非数据中心环境**（办公室/实验室）的可行性 |
| **网络** | ConnectX-8 双 400GbE 使多台 Super AI Station 可 **组成小型超节点集群** |
| **I/O** | RTX PRO 卡独立配置说明 GB300 **无图形输出能力**（纯计算加速器设计）——与 HGX 一致 |
| **竞品** | 对标 AMD Radeon Pro W7900 双路工作站、Intel Xeon 6 + Max GPU 开发者平台 |

> 🔑 **关键判断**: Super AI Station 的出现意味着 **超节点正在从机架级下沉到桌面级**。虽然 1,600W TDP 需要专用电路，但它让 GB300 的开发/调试/小型部署脱离了数据中心基础设施依赖。对于 AI 初创公司和研究团队来说，这是目前唯一能以 $125K 获得 GB300 性能的途径。

## ② 🟢 **Wiwynn + TE Connectivity 800V DC 液冷供电架构——下一代 AI 集群电力基础设施**

**时间**: 2026-06-26 · **来源**: ServeTheHome（Patrick Kennedy）
**原文标题**: 「Liquid-Cooling a TE Connectivity 800V DC Busbar and More from the Wiwynn Booth」
**链接**: https://www.servethehome.com/liquid-cooling-a-te-connectivity-800v-dc-busbar-and-more-from-the-wiwynn-booth/

### 概述

Wiwynn 在 Computex 2026 展出了 **下一代 AI 服务器的 800V DC 供电架构**，与 TE Connectivity 联合展示。随着新世代 AI 加速器的功耗密度持续攀升，数据中心正在从传统 48V/400V 向 **800V DC 配电** 转型，且 **busbar 本身也开始液冷化**。

### 供电架构全景

```
     数据中心配电演进
     ┌──────────────────────────────────────────────────┐
     │  传统: 480V AC → PSU → 12V/48V → 服务器          │
     │  ORv3: 48V DC → 机架内 DC-DC → 服务器            │
     │  下一代: 800V DC → 液冷busbar → 模块化 DC-DC     │
     │           ↓                                       │
     │          busbar 本身需要液冷散热                   │
     └──────────────────────────────────────────────────┘
```

### HVDC IT GEAR TO ELCON MICRO+ Power Cable Assembly

| 维度 | 规格 |
|:-----|:-----|
| **输入侧** | **800V / 75A** HVDC IT gear 端 |
| **输出侧** | **400V** ELCON MICRO+ 端 |
| **形态** | 预组装线缆模组，**整条装入**（非产线手工组装） |
| **定位** | 在服务器内部两个 **100+ cm² 加速器模块** 之间走线 |
| **制造** | 支持 **机器人自动化装配**——整条 drop-in |

### 液冷化 busbar

> 💡 **核心创新**: 800V DC busbar 本身需要液冷散热——因为在高电流密度下，busbar 的焦耳热已经无法靠自然对流或风冷排出。这是 **供电基础设施首次进入液冷回路**。

### 服务器内部布局

```
  前置面板             后置面板
  ┌─────────────────────────────────────────┐
  │ 管理模块 | E1.S ×4 | → 光网络 dummy ← │
  │                                         │
  │         ┌────────────────────┐          │
  │         │  加速器模组 A      │          │
  │         │  (100+ cm²)       │          │
  │         │                   │          │
  │         │  800V/75A 电源线  │          │ ← busbar 连接器
  │         │  装配总成 (中间)   │          │
  │         │                   │          │
  │         │  加速器模组 B      │          │
  │         │  (100+ cm²)       │          │
  │         └────────────────────┘          │
  │                    ↓                     │
  │          液冷喷嘴 + 机内 Scale-Up 网络    │
  └─────────────────────────────────────────┘
```

### 超节点意义

| 角度 | 分析 |
|:-----|:------|
| **供电密度** | 800V / 75A = **60kW/通道**，对比 48V 同电流仅 3.6kW——密度提升 **16.7×** |
| **散热挑战** | 高电流 busbar 首次进入液冷回路，意味着超节点的 **功率密度瓶颈已从芯片扩展到配电基础设施** |
| **制造自动化** | 整条电源线模组化「drop-in」设计 → 支持机器人装配，是 **超大规模部署的可制造性前提** |
| **Scale-Up 网络** | 机内已有 scale-up 网络液冷接头（暗示 NVLink/UALink 也在液冷路径内） |
| **时间节点** | 展示为性能/功耗模型（dummy 加速器模块），预计配合 **2027 年 Vera Rubin 超节点级产品**实用化 |

> 🔑 **关键判断**: 液冷 busbar 是 **所有 2027+ 超大型 AI 集群无法绕过的技术节点**。当单机柜功耗突破 200kW（NVIDIA Vera Rubin NVL4 机柜），busbar 本身的散热不再是可选特性，而是刚性约束。Wiwynn + TE 的展示表明整个供应链已开始布局这条技术路线。
