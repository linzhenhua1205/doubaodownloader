### 🟢 **NVIDIA Vera Rubin 平台正式发布** — 为科学计算打造的世界级超节点

**来源**: NVIDIA Newsroom 新闻稿，2026-06-22
**标题**: 「NVIDIA Vera Rubin Delivers World-Class Supercomputers for Science」
**链接**: NVIDIA Newsroom

| 维度 | 内容 | 超节点意义 |
|:-----|:-----|:----------|
| **FP64 能力** | 原生双精度 FP64 性能（非模拟） | 填补科学计算/HPC 对 GPU 双精度性能的刚需 |
| **CUDA-X 库** | 完整 CUDA-X 加速库栈 | 保证从传统 HPC 到 AI 的无缝迁移 |
| **HPE Cray GX5000 架构** | 基于 HPE Cray Supercomputing GX5000 | 已验证的超大规模互联架构（Cray Slingshot-11 升级） |
| **Quantum-X800 IB** | NVIDIA Quantum-X800 网络 | 第八代 InfiniBand，400Gbps/端口 |
| **客户合同** | LANL、LRZ Blue Lion、NERSC Doudna 等已签约 | 主要国家实验室和超算中心 |

#### 超节点架构设计

| 层次 | Vera Rubin 配置 | 说明 |
|:-----|:---------------|:-----|
| **GPU 节点** | NVIDIA Rubin GPU + Vera CPU 组合 | 与大语言模型训练/推理场景共用 |
| **纯 CPU 节点** | **2,300 颗独立 Vera CPU** (Mission) + **~1,150 颗** (Veritas) | 仅 HPE Cray GX240 blade 可插拔 |
| **互联** | NVLink-C2C + Quantum-X800 InfiniBand | CPU↔GPU 相干 + GPU↔GPU/跨机架 |
| **单机架密度** | 144 GPU / rack（NVL4 形态） | 维持超节点级密度 |
| **出货时间** | **Q4 2026** | 5× OEM 方案同步交付 |

> 💡 **关键信号**: Vera Rubin 的 **双线架构**（GPU 节点 + 独立 Vera CPU 节点）是超节点设计的重大变化：首次在一套超节点系统中同时部署两种计算形态，通过统一网络互联。这针对的是 **科学计算+Agentic AI 融合工作负载**——需要大量纯 CPU 节点运行模拟/AI agent 编排，GPU 节点加速训练/推理。

## ② 🟢 **LANL Mission / Vision / Veritas：最大超节点系统首次完整披露**

**来源**: NVIDIA Blog (Chris Porter)，2026-06-22
**标题**: 「NVIDIA Vera CPU Opens the Way for Agentic Scientific AI at Los Alamos National Laboratory」

### 三系统架构全景

| 系统 | 节点组成 | 定位 | 上线 | 
|:----|:---------|:-----|:-----|
| **Mission** | Vera Rubin GPU 节点 + **2,300** 颗独立 Vera CPU（GX240 blade） | NNSA ASC 第五代超级计算机，替代 Crossroads | **2027** |
| **Vision** | Vera Rubin GPU 节点 + 配套 CPU 节点 | 基础科学研究（材料/核科学/能源/生物医学/AI） | **2027** |
| **Veritas** | ~1,150 颗独立 Vera CPU + Vera Rubin 节点 | LDRD 项目，Agentic AI for Science 先行验证 | 同 Mission/Vision 时间线 |

### Vera CPU 在超节点中的作用

| 维度 | 数据 | 对比基准 |
|:-----|:-----|:--------|
| **URSA Agent 性能** | **7×** CPU 性能提升 | vs Crossroads x86 CPU |
| **Branson 模拟性能** | **>3×** | vs Crossroads x86 CPU（Monte Carlo 传热模拟） |
| **单 CPU v.s. 单 x86 Socket** | **>3×** 性能 + **4× 每核内存** + **6× 每节点内存** | vs Crossroads x86 |
| **核心架构** | 定制 Olympus core + LPDDR5 + 片上高速互联 | 非标准 Arm 核 |
| **应用场景** | AI Agent 推理、Monte Carlo 模拟、仿真编排 | 科学计算 + Agentic AI |

> 💡 **关键洞察**: LANL 的 Mission/Vision/Veritas 三系统设计本质上是 **「超节点 + CPU 扩展层」的混合架构**。Vera CPU 独立节点专门负责 Agent 推理和经典模拟，GPU 节点负责训练和大规模并行计算。这为未来超节点设计提供了 **「GPU-centric + CPU Agent 扩展」的新范式**。

### URSA Agent：超节点级 AI for Science 的软件层

| 维度 | 内容 |
|:-----|:-----|
| **全称** | Universal Research and Scientific Agent |
| **能力** | 提出假设 → 选择工具 → 启动模拟 → 分析输出 → 优化下一步 |
| **架构** | 模块化、反馈驱动的 AI 框架（Venado 上原型 → Mission/Vision 扩展） |
| **意义** | Agentic AI 从云端进入超节点级超算环境 |
