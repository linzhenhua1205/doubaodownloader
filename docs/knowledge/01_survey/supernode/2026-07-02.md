## 2️⃣ AMD Versal HBM→LPDDR5X 被迫转向：HBM 供给危机向外蔓延

- **来源**: [ServeTheHome](https://www.servethehome.com/amd-pivots-from-hbm-to-lpddr5x-for-new-versal-premium-gen-2-memory-on-package-chips/) (Ryan Smith, 2026-06-30)
- **分类**: 🔧 超节点内存架构 / HBM 供应链

### 核心事实

| 时间节点 | 事件 |
|:---------|:-----|
| **2018** | Xilinx 推出 Virtex UltraScale+ HBM FPGA（首款搭载 HBM 的可编程逻辑产品） |
| **2023** | Versal HBM 系列发布，搭载 **32GB HBM2e** |
| **2025年9月** | **Versal HBM 系列宣布停产** — 因 HBM2e 供应完全枯竭 |
| **2026年7月1日** | **最后订单日** — 从此 AMD 可编程逻辑不再供应任何含 HBM 的产品 |
| **2026年6月30日** | Versal Premium Gen 2 Memory on Package 发布 — **LPDDR5X 取代 HBM** |

### HBM 供给危机的结构原因

| 原因 | 详细说明 | 量化数据 |
|:-----|:---------|:---------|
| **AI GPU 需求挤出** | HBM3/HBM4 产线被 NVIDIA+AMD Instinct 完全包揽 | SK hynix/Samsung/Micron 全部 HBM3 产能已签约至 2027+ |
| **HBM 代际更迭加速** | HBM2e→HBM3→HBM3E→HBM4 间隔仅 2-3 年 | 非 GPU 产品无法跟上更换节奏 |
| **FPGA 长生命周期冲突** | Xilinx/AMD FPGA 正常产品周期 15-20 年（UltraScale+ 支持至 2045） | HBM 供货周期远短于 FPGA 需求周期 |
| **只有三大供应商** | SK hynix / Samsung / Micron | 无第二梯队，议价权完全倾斜 |

### AMD 的被迫应对

> **"HBM is out, and LPDDR5X is in."**

| 维度 | 原方案 (HBM) | 新方案 (LPDDR5X) |
|:-----|:------------|:-----------------|
| 内存类型 | HBM2e (已停产) → HBM3 (不可获取) | **LPDDR5X** (标准供应充足) |
| 内存容量 | 32GB (HBM2e, Versal HBM) | 未披露（LPDDR5X on package） |
| 内存带宽 | ~1.6 TB/s (HBM2e) | 显著低于 HBM，但适应 FPGA 工作负载 |
| 供应可靠性 | ❌ 完全依赖 HBM 产线分配 | ✅ 消费级 LPDDR5X 供应弹性大 |
| 产品生命周期 | ❌ HBM 供货周期 < 5 年 | ✅ LPDDR5X 供货周期 > 10 年 |
| 市场影响 | AMD 失去 FPGA HBM 产品线竞争力 | 转向更保守但可持续的方案 |

### 对超节点/AI 集群架构的启示

1. **HBM 供给的"挤出效应"已超出 GPU 领域**: 任何需要 HBM 的非 GPU 加速器（FPGA/SmartNIC/DPU/类脑芯片）都在承受 HBM 供给收缩的压力
2. **超节点系统中 HBM 分配优先级**: 在 HBM 供给瓶颈下，超节点整体架构设计需优先保证 GPU 的 HBM 分配，非 GPU 组件（如用于 AI 推理加速的 FPGA 卡）可能需要寻找 LPDDR5X/GDDR7 等替代方案
3. **LPDDR5X on Package 成为备选范式**: AMD 的 LPDDR5X on Package 方案可能被其他受 HBM 挤出的芯片设计者效仿
4. **对芯片设计路线的长远影响**: 如果 HBM 供给约束长期持续，超节点中"非 GPU 加速器"的设计路线可能从 "追求极致带宽(HBM)" 转向 "足够带宽(LPDDR5X)+足够的芯片内缓存"

### 战略解读

> 这不仅是 AMD 可编程逻辑部门的产品策略调整，更是 **AI 基础设施 HBM 供给紧张的结构性证明**。当 FPGA 这种非 GPU 产品连 HBM2e 这种"前代"产品都无法获得时，说明 HBM 供应链已经完全被 AI GPU 锁死。对超节点设计而言，**非 GPU 组件的内存架构选择必须考虑 HBM 不可得性这一长期约束**。
