## 📌 SSD 产品

### 🏆 Kioxia XG10 Series — PCIe Gen5 高性能客户端 SSD

| 维度 | 数据 |
|:-----|:------|
| **发布时间** | 2026-05-13 |
| **来源** | [ServeTheHome - Cliff Robinson](https://www.servethehome.com/kioxia-xg10-series-pcie-gen5-ssds-announced/) |
| **接口** | PCIe Gen5 x4 NVMe |
| **容量** | 512GB / 1TB / **2TB / 4TB** |
| **NAND** | 512GB-1TB: BiCS FLASH Gen6 TLC；**2TB-4TB: BiCS FLASH Gen8 TLC（CBA 技术）** |
| **顺序读写** | 最高 **14 GB/s 读 / 12 GB/s 写** |
| **随机性能** | **2M 随机读 IOPS / 1.6M 随机写 IOPS** |
| **控制器** | 8 通道 SoC 设计（**含 DRAM**） vs BG8/EG7 的 4 通道 DRAM-less |
| **功耗** | Active: **10W**（高于 BG8 的 5W 和 EG7 的 4.5W）|
| **安全** | TCG Opal 2.02 Self-Encrypting Drive |
| **定位** | 高性能客户端 / AI PC / 工作站 / 高端游戏 |
| **意义** | Kioxia 形成了完整的客户端 SSD 三段产品矩阵：EG7（QLC 入门）→ BG8（TLC 主流）→ XG10（TLC 旗舰+DRAM）。CeBIT 2026 之前已完成产品线全规格覆盖 |

## 📌 市场动态

### 🏆 NAND 短缺恶化预期

| 维度 | 数据 |
|:-----|:------|
| **来源** | Silicon Motion CEO 在 Computex 2026 访谈 / Tom's Hardware |
| **核心预判** | NAND wafer 供应短缺 **2027 年将更加严重** |
| **原因** | (1) HBM 产能侵占 3D NAND fab 空间；(2) AI 服务器 SSD 需求暴增；(3) NAND 价格过低导致供应商投资犹豫 |
| **影响** | SSD 价格可能保持高位或继续上行；控制器出货量虽创新高，但 NAND 颗粒供应为瓶颈 |
