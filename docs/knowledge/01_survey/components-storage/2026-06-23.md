## 🏆 头条二：Kioxia CD9P-R 企业级 PCIe 5.0 SSD 全面评测 — BiCS 第8代 TLC 首秀

**来源**: StorageReview, June 18 2026

Kioxia 发布了新一代数据中心 NVMe SSD **CD9P-R 系列**，这是 Kioxia CD 系列首款采用 **BiCS FLASH 第8代 TLC** 的 SSD：

| 规格 | 数值 |
|:-----|:-----|
| 接口 | **PCIe 5.0 x4, NVMe 2.0** |
| 最大容量 | E3.S: 30.72TB / **2.5-inch: 61.44TB** |
| 峰值顺序读 | **14,800 MB/s**（7.68TB/15.36TB 型号） |
| 随机读 | **2.6M IOPS** |
| 随机写 | **450K IOPS**（7.68TB/15.36TB，较上代 CD8P-R 200K **提升 2.25×**） |
| 顺序读提升 | 12,000→14,800 MB/s（**+23%**） |
| 功耗 | 23W typ（仅微增 2W） |
| DWPD | 1（读密集型） |
| NAND | KIOXIA BiCS FLASH **Gen 8** TLC（7.68TB-30.72TB） |
| 兼容 | OCP Datacenter NVMe SSD v2.5（部分） |

**评测亮点与不足**:

✅ **读性能顶尖**
- 128K 顺序读实测 **14,235.9 MB/s**，与 Pascari X200P 并列第一梯队
- **4K 随机读 QD1 延迟仅 ~30µs**，约为竞品（60-90µs）**一半**——在低队列深度下具有显著优势
- **GPU Direct Storage (GDS) 1M 块顺序读达 6.2 GiB/s**，全场最高

✅ **负载一致性出色**
- DLIO 检查点测试（LLAMA 3.1 405B）稳定在 572s/pass，未出现 Pascari X200P 的 674s 峰值摆动
- 属于"最稳定的一批"

⚠️ **写性能中游**
- 128K 顺序写 6,912 MB/s 垫底（读密集型定位使然）
- 4K 随机写峰值 1,263K IOPS，处于中下水平

**意义**: CD9P-R 是 BiCS 第8代 NAND 在数据中心 SSD 上的首秀，**2.25× 随机写提升** 和 **~30µs 低队列读延迟** 展示了第8代 BiCS 在延迟控制方面的实质性进步。7.68TB 为"甜点容量"。

## 🏆 头条三：HighPoint Rocket 1604L — Astera Labs Retimer 驱动的 Gen5 x16 四槽 SSD 卡评测

**来源**: StorageReview, June 19 2026

HighPoint 发布了 **Rocket 1604L**（$399），采用 **Astera Labs PT5161LRS retimer** 架构的 Gen5 x16 四 M.2 扩展卡：

| 指标 | 数值 |
|:-----|:-----|
| 价格 | **$399** |
| 架构 | **Astera Labs PT5161LRS retimer**（非 PCIe Switch，无 RAID） |
| 每槽带宽 | 4× Gen5 x4（独立通道） |
| 测试配置 | 4× Samsung 9100 PRO 4TB |
| **128K 顺序读聚合** | **55.6 GB/s**（~94% 理论峰值） |
| 4K 随机写 | **10.1M IOPS**（~97% 理论峰值） |
| 4K 随机读 | **8.83M IOPS**（~100% 理论峰值） |
| 延迟 | 顺序读 604µs avg / 906µs p99 |
| 功耗 & 散热 | 全铝散热+风扇，60s 持续无降频 |

**架构亮点**:
- **retimer 路线 vs PCIe switch 路线**：比被动分叉卡更稳定的信号完整性（Gen5 32GT/s 高频信号关键要求），又比 switch 方案（$700+）便宜且无额外延迟
- **前提**: 主机平台须支持 x4/x4/x4/x4 bifurcation（Threadripper TRX50/WRX90, Xeon W, EPYC）

**意义**: 在 Gen5 SSD 飙速时代，retimer 方案正在成为 PCIe 扩展卡的主力架构，而 Astera Labs 继 CXL 内存池化芯片后，在**存储扩展 retimer 市场也建立了关键地位**。

## 📌 核心判断

1. **BiCS Gen8 首秀合格**: Kioxia CD9P-R 证明第 8 代 BiCS TLC 在**延迟控制和读性能**上已进入第一梯队，2.25× 随机写提升是最大亮点，但顺序写仍为读密集型定位所限
2. **Gen5 扩展卡 retimer 路线成熟**: HighPoint 1604L + Astera Labs 组合证明 retimer 方案可在 Gen5 信号完整性和成本之间取得最佳平衡
3. **AI 定价危机蔓延至消费层**: Prime Day "最后好价" 叙事验证了 AI 驱动的 NAND 涨价已从数据中心传导至消费 SSD 市场
4. **存储厂商 AI 管道化加速**: Everpure Data Stream + KV Cache Accelerator 揭示企业存储正在从"块/文件存储"向"AI 数据管道"进化
