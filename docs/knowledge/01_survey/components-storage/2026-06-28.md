- Micron 已签署 **16 个 SCA**（覆盖数据中心、消费、汽车三大段）

- 最小承诺价格模式下，14/16 个 SCA 累计约 **$100B**
- 预计最终 SCA 完全签署后，**半数以上营收**将处于合约框架下
- **约 40%** 的 DRAM 营收将带固定价或当前 CQ2 水平上限价

### 5️⃣ 💾 其他值得关注的信息

#### 5.1 DDR2 合约价飙升 60%（2003 年时代的内存）

**Tom's Hardware（2026-06-22）**：

- **DDR2**（2003 年推出的标准）Q2 合约价上涨 **55%~60%**
- 预计 **Q3 再涨 35%~40%**
- 原因：老旧工业/嵌入式设备仍需 DDR2 维护库存，需求未减而产能早已被转产至 DDR5/DDR4
- 示例性说明整个存储产业链的**全面供应紧绷**

#### 5.2 AMD 收购 MEXT 获 Predictive Memory Engine 技术

**Tom's Hardware（2026-06-15）**：

- AMD 收购 **MEXT**，获得其 **Predictive Memory Engine**（PEM）
- 核心技术：在 DRAM 和 NAND 之间智能调度冷/热数据，**让 Flash 对应用层展现为 DRAM**
- 属于 Memory Tiering 领域的硬件+软件方案
- 对 AMD EPYC 的数据中心内存分层能力至关重要

#### 5.3 Qualcomm HBC（Hybrid Bonding Cube）近内存架构

**Tom's Hardware（2026-06-25）**：

- Qualcomm 发布 **HBC（Hybrid Bonding Cube）** 近内存 AI 架构
- **AI250 / AI350 加速器**首次采用该架构
- 声称已"**打破内存墙**"（breaking the memory wall）
- 通过混合键合（Hybrid Bonding）技术实现计算单元和内存的紧密耦合，减少数据搬运延迟

#### 5.4 Apple CEO Tim Cook 预警 AI 驱动的涨价不可避免

**Tom's Hardware（2026-06-18）**：

- Tim Cook 承认 Apple 无法再"**shield**"客户免受 DRAM 和 NAND 价格上涨影响
- 全系 iPad/MacBook 因存储成本已提价
- Apple 作为全球最大消费电子存储买方之一，其表态标志着**终端传导效应正在加速**

#### 5.5 行业联盟敦促白宫：AI 数据中心的内存消耗正在威胁其他行业

**Tom's Hardware（2026-06-05）**：

- 多家行业组织联合敦促 Trump 政府采取紧急行动
- 核心担忧：AI 数据中心对 DRAM/NAND 的**极端消耗**正在排挤其他行业
- 尽管美国已在半导体本土供应链投资数十亿美元，风险仍在加剧
- Eventbrite CEO 此前在公开信中尖锐批评：$1.5B+ 参数模型的 GPU 需求正扼杀互联网创新
