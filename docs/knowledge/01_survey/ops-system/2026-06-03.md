## 1️⃣ CNCF 生态：云原生 → AI Native 范式转变

### 🌐 "Cloud native is now AI-native" (CNCF Staff Post, Jun 2)
KubeCon EU 2026 圆桌讨论：AWS/Google/Microsoft/solo.io 专家共识
- **AI 生产就绪三要素**：平台成熟度 + 安全设计 + 社区贡献
- **Kubernetes AI Conformance** 成为互操作性标准
- AI 工作负载是巨大的"单体"——标准 K8s 难以满足紧耦合需求
- **关键 K8s 演进方向**：
  1. **Pod Groups (Workload API)** — 将 Pod 组视为单一故障域
  2. **Dynamic Resource Allocation (DRA)** — GPU/专用芯片调度
  3. **Inference Gateways** — Gateway API 标准化 AI 网关
- **Agentic SRE 兴起**：AI 代理辅助根因分析，人类保留关键决策权
- 安全延伸至模型供应链 + 非确定性输出风险

### 🎯 KEDA GPU 自动缩放 (CNCF Community Post, May 27)
- KEDA + NVML 构建 GPU 感知外部缩放器
- **GPU 指标**：SM 利用率 / 内存利用率 / VRAM 使用率 / 温度 / 功耗
- **预置 Profile**：vLLM(80% memory), Triton(75% gpu_util), training(90% gpu_util), batch(70% memory)
- 支持 **scale-to-zero**
- DaemonSet 模式规避 CGO 限制
- **深度跟踪 🏷️** — GPU 感知 HPA 是 K8s AI 化的关键能力

### ⚡ NetEase Games: 30秒 LLM 冷启动 (CNCF End User Post, May 21)
- "弹性计算只有在数据能同样快速移动时才有用"
- K8s 上 LLM 推理的冷启动优化

## 3️⃣ NVIDIA AI 工厂运维体系

### 🏭 NVIDIA DSX OS (NVIDIA Blog, May 31)
- 开放模块化 AI 工厂操作系统，Gigawatt 级规模
- **DSX Exchange**：MQTT 桥接 IT/OT，将电网事件/热数据/功率异常暴露给管理层
- **MCP Server**：为 AI Agent 提供统一的工具目录
- **MaxLPS**：动态电源策略，**+40% GPU** 同功率预算下
- **DSX Flex**：将工作负载连接到电网服务（需求响应/负荷削减/可再生能源）
- **NVIDIA Infra Controller (NICo)**：API 驱动的裸金属生命周期管理 + BlueField DPU 硬件隔离
- **AI Cluster Runtime (AICR)**：版本锁定运行时配置，消除配置漂移
- **NV Sentinel**：K8s 原生 GPU 故障检测，秒级自动修复
- **Fleet Intelligence**：全局 GPU 舰队可视性（已上传截图）
- **深度跟踪 🏷️** — NVIDIA 打造 AI 工厂"操作系统"的完整战略布局

### 🖥️ NVIDIA Fleet Intelligence (NVIDIA Blog, May 11)
- 实时 GPU 舰队可视性和优化
- Lambda 是早期采用者

### 📊 GPU Usage Visibility on K8s (NVIDIA Blog, May 21)

## 📊 趋势判断

| 方向 | 热度 | 关键信号 |
|:-----|:----|:---------|
| AI Native K8s | 🔥🔥🔥🔥🔥 | Pod Groups/DRA/Inference Gateways 三大演进方向 |
| OTel 生态成熟 | 🔥🔥🔥🔥🔥 | 正式毕业 + GenAI 可观测性 + Jaeger Agent 追踪 |
| AI 工厂操作系统 | 🔥🔥🔥🔥🔥 | DSX OS 整合 IT/OT/MCP/Fleet/NV Sentinel |
| GPU 自动缩放 | 🔥🔥🔥🔥🔥 | KEDA GPU Scaler 开源，NVML 驱动 |
| Prometheus 现代化 | 🔥🔥🔥 | 存储层升级 + UX WG + OTel 集成深化 |

> **核心判断**：运维范式的根本转变 — 从"监控 AI 工作负载"到"AI 原生运维"。K8s 正在为 AI 重构（Pod Groups/DRA），OTel 覆盖 Agent 追踪，NVIDIA 构建完整的 AI Factory OS。AIOps 不再是附加能力，而是基础设施的基础组成。
