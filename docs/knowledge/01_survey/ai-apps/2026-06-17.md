## ① 🚀 SpaceX $600亿全股票收购 Cursor（6/16, TechCrunch）

- **来源**: [TechCrunch](https://techcrunch.com/2026/06/16/spacex-to-acquire-cursor-for-60b-in-stock-days-after-blockbuster-ipo/) | Jun 16 | Sean O'Kane
- **来源**: [Ars Technica](https://arstechnica.com/ai/2026/06/spacex-to-acquire-ai-coding-platform-cursor-for-60-billion/) | Jun 16 | Samuel Axon
- **来源**: [The Verge](https://www.theverge.com/ai-artificial-intelligence) | Jun 16 | Robert Hart

**关键数据**:
- SpaceX 以 **$600亿全股票** 收购 AI 编程平台 Cursor（Anysphere），预计 Q3 交割
- 此为距今 **不到 2 个月内** 双方第二次接触（4月首次宣布 $600亿购股或 $100亿分手费）
- Cursor 正进行 **$20亿融资轮**（a16z/Thrive/NVIDIA 参与），投后估值 $500亿
- 此前 Cursor 累计融资：2025年6月 **$9亿 Series C** + 2025年末 **$23亿**
- 收购背景：xAI **11 位联合创始人全部离职**，Musk 公开承认"xAI 第一次没建对，正在从地基重建"
- SpaceX IPO 后股价从 $135 飙升至 **$200+**，短短几天增加近 **$1万亿** 市值（≈16个Cursor）
- SpaceX 对投资人的 TAM 叙事：**$28万亿** 总市场，其中 **$26万亿** 靠 AI 部门

**行业意义**: 史上最大 AI 编程工具收购案。Cursor 在 AI Coding 领域曾以 $290亿 估值冲刺，如今以 $600亿 成为 SpaceX AI 战略核心拼图。

## ③ ⏸️ Anthropic 暂停 Agent SDK Token 计费计划（6/16, Ars Technica）

- **来源**: [Ars Technica](https://arstechnica.com/ai/2026/06/anthropic-pauses-token-based-billing-for-its-claude-agent-sdk/) | Jun 16 | Kyle Orland

**关键数据**:
- 原计划 6月15日生效：Agent SDK 使用（包括第三方 App 和 `claude -p` 命令）按 API 费率单独计费
- 开发者测算：Claude Opus 用户每天 **2-3 条消息** 即回本订阅费，重度用户每月可薅出 **数倍 API 价值**
- 编辑器 Zed 开发者警告："对于重度使用 Agent 的人，这是重大成本上涨"
- Anthropic 在生效当天 **急刹车**，更新计费页面为"暂停变更，正在优化方案"
- 背景：此前 GitHub Copilot 已实施 Token 计费引发 **大规模用户反弹**
- Anthropic 正准备 IPO，需平衡开发者好感与财务可持续性

| 调整后净亏损（不含一次性） | — | **~$80亿** |
