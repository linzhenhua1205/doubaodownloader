## 2️⃣ 💬 "Chat is dead" — OpenAI 准备最大规模的 ChatGPT 超级应用改造

- **来源**: [Ars Technica](https://arstechnica.com/ai/2026/06/08/chat-is-dead-openai-preps-overhaul-of-chatgpt/) via Financial Times | 2026-06-08
- **核心内容**:

### 战略转型：ChatGPT 将从聊天机器人变为超级应用

- **"Chat is dead"** — 一位 OpenAI 高级员工的原话
- ChatGPT 将彻底改造为 **"超级应用"（super app）**，融合：
  - **Codex 编程**（核心增长引擎）
  - **AI Agent 能力**（自动完成任务，从 booking 到日程管理）
  - **图像生成**
  - **第三方合作伙伴应用**（Canva、Booking.com 等）
- 新界面将引导用户从纯对话转向 **coding、image-gen、partner apps**，最终目标是**模型自动理解用户意图，无需显式 prompts**

### 关键数据

| 指标 | 数据 |
|:-----|:-----|
| **Codex 周活用户** | **500 万+**（自 2 月桌面版发布增长 **6 倍**） |
| **企业客户数** | **200 万** 企业客户 |
| **企业收入占比** | **40%** → 预计 **2026 年底达 50%** |
| **ChatGPT 用户总量** | **近 10 亿** |
| **资源调整** | 已关闭 Sora 视频产品、砍掉 ChatGPT 内购功能、将资源转向企业级产品 |

### 内部重组

- ChatGPT、Codex 等产品团队已整合至 Thibault Sottiaux（原 Codex 负责人，现 OpenAI 核心产品与平台总负责人）统一领导
- 原产品负责人 Kevin Weil 等高管已离职
- 战略重心：从「消费者应用」全面转向「企业 Agent 平台」

- **意义**: 这是 OpenAI 从 **"AI 聊天公司"向"AI 操作系统公司"的彻底转型**，与 Anthropic 的战略从两端向中间收敛（"Anthropic 先赚钱，OpenAI 先打品牌，现在两者在 IPO 压力下走向一致"）

## 3️⃣ 🍎 Apple AI 竟然跑在 Nvidia 硬件上 — WWDC 最大意外

- **来源**: [The Verge](https://www.theverge.com/ai-artificial-intelligence) WWDC 2026 现场报道 | 2026-06-08
- **关键披露**:
  - Apple 在 WWDC Tech Talk 中透露：**Apple Foundational Model 运行在 Nvidia 硬件上**（通过 Google Cloud）
  - Apple 与 **Nvidia、Google、Intel** 三家合作搭建 Private Cloud Compute（PCC）基础设施
  - **这是一个历史性反转**：Apple 曾是 Nvidia 的最大批评者之一（多年来因功耗/散热问题拒绝采用 Nvidia GPU，一直使用自己的芯片或 Google TPU）
  - **意义**: **AI 训练/推理的算力需求如此之大，以至于连 Apple 都不得不放下成见使用 Nvidia 硬件**——这是 AI 算力霸权格局的终极确认

### WWDC 2026 Apple Intelligence 完整发布矩阵

| 产品 | 要点 |
|:-----|:------|
| **Siri AI** | 全新对话式语音助手，有独立 App，秋季英语版首发 |
| **iOS 27** | Siri AI 原生集成 |
| **Image Playground** | 大幅改善，不"烂"了 |
| **Shortcuts** | AI 工作流构建能力 |
| **Siri in Camera** | AI 分账功能解决聚餐算账痛点 |
| **Safari 扩展** | 用户可"vibe-code"浏览器扩展 |

### EU 延迟争议

- Apple 以 **DMA 合规为由**，表示 Siri AI 在 EU 不会立即上线：仅限 macOS/watchOS/visionOS
- Apple 表示在 EU 的 DMA 要求下，无法确保用户隐私保护

- **趋势判断**: Apple 从"端侧隐私 AI"全面转向"端侧 + 云端"混合路线，且云端的 Nvidia 依赖是短期不可逆转的。这与 Apple 之前塑造的自信形象形成鲜明对比

## 📈 趋势判断

1. **OpenAI 的"超级应用"时刻** — "Chat is dead"不是一个噱头，而是 OpenAI 从聊天工具向 Agent 操作系统的正式转身。与 Anthropic 战略趋同说明：**AI 的终局是 Agent 平台，不是聊天机器人**
2. **AI IPO 三巨头同台竞技** — OpenAI + Anthropic + SpaceX 三家公司同在 2026 年进入公开市场，这将创造 **有史以来最大的科技 IPO 窗口**，也意味着 AI 行业从"烧钱竞赛"转向"盈利叙事"
3. **Apple 的 Nvidia 依赖暴露 AI 算力霸权** — 即使 Apple 也会"真香"，Nvidia 在整个 AI 生态中的不可替代性达到顶峰。Apple 的端侧 + Nvidia 云端混合路线将成为行业标配
4. **AI Agent 成为安全攻击新战场** — npm 凭证窃取事件只是开始，Agent 自动执行代码的特性天生是供应链攻击的理想目标，**AI Agent 安全将成为下一个爆发性产品品类**
