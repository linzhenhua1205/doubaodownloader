### 15.3 芯片设计数学的"二八定律"

> **20% 的数学分支覆盖 80% 的芯片设计日常**：

| 优先级 | 数学分支 | 覆盖岗位数 | 理由 |
|:-------|:---------|:----------|:-----|
| **P0** | 布尔代数/数字逻辑 | 全部数字岗 | RTL 编码→综合→验证的绝对基础 |
| **P0** | 图论（路径/树/划分） | 后端/STA/NoC/DFT | 物理设计的核心抽象语言 |
| **P0** | 概率与统计 | 全部 | 良率/SPC/测试/可靠性无处不在 |
| **P0** | 线性代数 | SPICE/RC提取/ML | 仿真、降阶、机器学习的共同基础 |
| **P1** | 微积分（偏导/PDE） | 模拟/器件/热/电磁 | 物理世界建模的语言 |
| **P1** | 数值分析 | SPICE/FEM/FDM | 所有仿真工具的理论基础 |
| **P1** | 傅里叶分析/信号处理 | DSP/SerDes/PLL | 频域分析的核心工具 |
| **P1** | 组合优化 | 后端/测试/DFT | EDA 工具的大脑 |
| **P2** | 复分析 | RF/传输线/控制 | 特定岗位需要 |
| **P2** | 信息论/编码 | ECC/FEC/通信 | 纠错和通信的基础 |
| **P2** | 排队论 | 架构/NoC | 性能建模 |
| **P3** | PDE + 固体力学 + 热流体 | 封装/散热/可靠性 | 特定方向深入 |
| **P3** | 有限域/数论 | ECC/加密 | 专门领域 |
