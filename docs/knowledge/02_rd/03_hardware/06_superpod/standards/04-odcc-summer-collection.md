### 1.1 ODCC 四层 AI 基础设施架构

```
INFRASTRUCTURE ─── Facility · Power · Cooling
    ┌── AIDC 供电: 800V HVDC Grid-to-Chip | SSCB/SSHCB 三级断路器
    └── 散热: 两相冷板 | 全液冷整机柜 | PUE 1.1

INTERCONNECT ─── Optical · Electrical · Protocol
    ├── Scale-Up: UALink 2.0 | CXL 4.0 | HBD 三档分级 (TB/s-class)
    ├── Scale-Out: UEC (LLR/CBCF/MRC) | RoCEv2 | 224G/448G NPO/CPO
    └── Optical: OCS MEMS Circuit Switch | SiPh OCS

HARDWARE ─── Compute · Storage · DPU · Switch
    ├── Compute: XPU Module (OAM/UBB) | ODS Disaggregated Units
    ├── Storage: NVMe SSD (PCIe 6.0/7.0) | CXL Memory Pool | JBOF
    └── Interconnect: 102.4T Switch | DPU | Retimer

MANAGEMENT ─── Orchestration · Ops · AIOps
    ├── Orchestration: K8s | Slurm | Unified Scheduler
    ├── Intelligent Ops: AIOps | CMDB+Monitoring | Fault Prediction
    └── Testing: Dora Auto-Test | Token Economics Simulator
```

## 4. 超节点专题 — ODS 开放解构超节点标准

> ⭐ **本届全会最大亮点**：ODS（Open Decomposed Supernode）标准深化定稿

### 4.1 ODS 标准基本信息

| 维度 | 内容 |
|:-----|:------|
| **全称** | 开放解构超节点技术规范 + 白皮书 1.0 |
| **牵头** | 中国移动（运营商立场） |
| **联合编制** | Intel、云尖信息（ODM） |
| **状态** | ✅ 深化定稿 + ✅ 白皮书 1.0 发布 |
| **定位** | 从架构层面定义「解构式」超节点的硬件单元划分、互联拓扑、管理接口规范 |
| **对标** | 相当于 OCP 的 OAM/UBB 规范在超节点尺度的扩展 |

### 4.2 ODCC 超节点标准发展时间线

```text
2025.12                         2026.03                         2026.06
   │                               │                               │
   ▼                               ▼                               ▼
超节点需求研讨          超节点标准启动编制         ODS 深化定稿 + 白皮书 1.0
(信通院牵头)             (20+ 家单位参与)          (15 项 AI 标准成果)
    │                               │                               │
    └── 五大层级框定 ──────────────→ 各层级细化 ──────────────→ 完整标准体系
        (芯片/互联/机柜/系统/运维)       (工作组分工)                 (73% 面向 AI)
```

### 4.3 ODS 核心框架

| 层级 | 解构单元 | 互联要求 | 管理规范 |
|:-----|:---------|:---------|:---------|
| **计算单元** | XPU Module (OAM/UBB 兼容) | Scale-Up: UALink/CXL | Redfish + OpenBMC |
| **互联单元** | Switch Tray (102.4T) | Scale-Out: UEC/RoCE | NVSwitch Mgmt |
| **存储单元** | NVMe JBOF / CXL Memory Pool | PCIe 6.0/7.0 | 存储管理 API |
| **管理单元** | BMC Module | OOB: NC-SI over 1GbE | IPMI + Redfish |
| **供电单元** | 800V HVDC BBU | I2C/PMBus | 供电遥测 |
| **散热单元** | 液冷 CDU/Rack | — | 冷却液管理 |
