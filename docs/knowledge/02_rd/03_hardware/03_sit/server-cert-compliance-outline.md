### 1.2 安规测试项 (IEC/GB 62368-1)

| 测试项目 | 条件 | 合格标准 | 超节点特殊要求 |
|:---------|:-----|:---------|:-----|
| **耐压测试(Hi-Pot)** | 1500V AC/1min(初级-地), 3000V AC/1min(初级-次级) | 无击穿/闪络 | 整柜级: 含液冷管路接地验证 |
| **绝缘电阻** | 500V DC | ≥100MΩ | 含水冷管路, 含冷却液绝缘 |
| **接地连续性** | 25A, ≤0.1Ω | 接地阻抗达标 | 各节点到大地点≤0.1Ω, 柜间接地≤0.2Ω |
| **漏电流** | 额定电压 | ≤3.5mA (Class I) | 72 GPU合计需分配验证 |
| **球压测试** | 75℃/1h | 压痕≤2mm | 冷板附近塑胶件, 需考核80℃ |
| **灼热丝测试** | 750℃/30s | 不起燃 | 连接器/绝缘材料扩展测试(GWFI≥850℃) |
| **温升测试** | 满载运行 | 各点≤额定温升 | 含全CUDA负载+液冷满流状态 |
| **异常工作测试** | 液冷失效/风扇全停/过载 | 无起火/无触电 | 液冷泵失效→GPU降频→安全关机链 |
| **机械强度** | 冲击/跌落/振动 | 安规距离不缩小 | 含管道QD抗冲击 |

### 1.3 安规设计要求 (整机+超节点)

**电气安全距离**:

| 绝缘类型 | 一次侧-地 | 一次侧-次级 | 适用场景 |
|:-----|:---:|:---:|------|
| **基本绝缘** | ≥1.5mm | ≥2.0mm | 塑胶外壳内电路 |
| **加强绝缘** | ≥3.0mm | ≥4.0mm | 金属外壳直达用户 |

**超节点额外要求**:
- 液冷管路与高压区域隔离(电气间隙加倍)
- Busbar绝缘: 48V与柜体≥1.5mm (基本绝缘), 800V与柜体≥5.0mm (加强绝缘)
- 漏水/冷凝水导电风险评估与对策
- 48V OR-ing防止单点故障引发电击风险

**防触电保护**:
- Class I设备: 金属外壳可靠接地(≤0.1Ω)
- X/Y电容放电回路: 断电1s内V≤60V (一次侧), V≤37V (SELV)
- 保险丝: √[t]特性匹配负载, 分断能力≥系统最大短路电流
- 过温保护: BMC监测+硬件温保开关双重保护

## 二、EMC认证

### 2.1 测试项目与标准 (完整版)

| 测试项目 | 标准 | 等级 | 限值/条件 | 超节点补充 |
|:---------|:-----|:----:|:-----|:-----|
| **辐射发射(RE)** | CISPR 32 / EN 55032 | Class A | 40dBμV/m@10m (30MHz-1GHz), 50dBμV/m@3m (1-6GHz) | 整柜级RE需暗室(3m或10m), GB200级需考量大开孔机柜 |
| **传导发射(CE)** | CISPR 32 / EN 55032 | Class A | 60dBμV@150kHz (QP) | 48V Busbar/800V HVDC 电力线CE |
| **静电放电(ESD)** | IEC 61000-4-2 | ±8kV/±15kV | B级(性能可自恢复) | 含液冷管路/QD/盲插面板 |
| **辐射抗扰度(RS)** | IEC 61000-4-3 | 3V/m 80MHz-6GHz | A级(不降性能) | GPU满载运行状态下测试 |
| **电快速瞬变脉冲群(EFT)** | IEC 61000-4-4 | ±2kV(电源)/±1kV(信号) | B级 | 48V/800V电力口 |
| **浪涌(Surge)** | IEC 61000-4-5 | ±2kV(线-线)/±4kV(线-地) | B级 | 800V HVDC需要到±6kV(线-地) |
| **工频磁场(PMF)** | IEC 61000-4-8 | 30A/m | A级 | — |
| **电压暂降/中断(DIP)** | IEC 61000-4-11 | 0%/70%/0% (不同时长) | B/C级 | BBU备份切换验证 |
| **谐波(Harmonics)** | IEC 61000-3-2 | Class A | 各次<限值 | 800V整流器谐波管控 |
| **电压闪烁(Flicker)** | IEC 61000-3-3 | Pst≤1.0, Plt≤0.65 | — | — |

### 2.2 EMC设计要求 (整机+超节点)

**机箱/机柜屏蔽**:
- 缝隙管控: ≤λ/20 (10GHz→≤1.5mm, 18GHz→≤0.8mm)
- 导电衬垫搭接阻抗≤10mΩ, 搭接点间距≤50mm
- 通风孔: 波导管阵列, 单孔最大尺寸≤5mm (12GHz截止)
- I/O面板: 所有连接器面板与机箱搭接, 弹片式EMI弹片

**电源滤波**:
- 电源入口: 共模扼流圈+X电容(0.1-1μF)+Y电容(2200-4700pF)
- DC-DC输入/输出: RC/LC滤波, π型滤波
- 48V Busbar: 端口滤波器, 差模100μH+10μF

**布线规则**:
- 高速信号走内层 (带状线), 紧邻参考地平面
- I/O线缆: 铁氧体磁环+屏蔽编织层
- 液冷管路: 金属编织屏蔽, 与机柜接地

**接地设计**:
- 多点接地: 高频优势, 各接地点间距≤λ/10
- 单点接地: 低频/模拟电路
- 混合接地: 数字多点+模拟单点+磁珠隔离
- 超节点柜体: 柜间等电位连接+建筑地

**超节点特殊EMC要求**:
- 72 GPU同步开关噪声 (SSO): 须评估同时性对传导/辐射的影响
- 液冷泵变频驱动器(VFD): 独立EMI滤波
- 800V HVDC整流器: IEC 61000-3-12谐波管控
- 柜间互联光纤: 无EMI贡献 (优势)

| 铅(Pb) | ≤0.1% | 高温焊料(≥85℃), 服务器BGA焊球 | BGA焊球豁免, 但需记录 |

## 五、网络安全与信息安全

### 5.1 服务器安全认证

| 认证 | 覆盖范围 | 要求 | 超节点额外要求 |

| 要求 | 内容 | 超节点实施 |
|------|------|------|
| **SBOM** | 软件物料清单 (SPDX/CycloneDX格式) | 含BMC/BIOS/CPLD/GPU固件/Switch OS |
| **HBOM** | 硬件物料清单 | 芯片+板卡+模块溯源 |
| **固件签名** | 所有固件须数字签名, 验签方可执行 | 逐级: RoT→UEFI→BMC→GPU→DPU→Switch |

### 5.3 超节点安全架构

```
RoT (DC-SCM 2.0)
 │
 ├─ SPDM验证: GPU ×72, NVSwitch ×9, DPU ×8, PCIe Switch ×4

 │

 │    RoT → UEFI FW → GPU FW → NCCL FW → DPU FW

```

## 六、认证测试流程与周期

### 6.1 标准服务器认证时间线

```
Phase 1 (Week 1-4):  预测试/自测
  ├─ EMC预扫 (近场/GTEM)
  ├─ 安规预测试 (耐压/接地/漏电流)
  ├─ 能效预测试 (80Plus预测)
  └─ 问题识别与整改

Phase 2 (Week 5-8):  正式认证
  ├─ 送样到认可实验室
  ├─ 安规全项测试 (2-3周)
  ├─ EMC全项测试 (2-3周)
  └─ 能效测试 (1周)

Phase 3 (Week 9-10): 整改+补测
  ├─ 不合格项分析
  ├─ 整改措施实施
  └─ 补测验证

Phase 4 (Week 11-12): 证书获取
  ├─ CCC: 工厂审查 (含)
  ├─ UL: 季度/年度工厂跟踪
  └─ CB: 报告转各国认证
```

| 认证 | 总周期 | 是否需要工厂审查 |
|------|:---:|:---:|
| **CCC** | 8-12周 | ✅ 初始+年度 |
| **UL** | 6-10周 | ✅ 季度跟踪 |
| **CE** | 4-8周 | ❌ 自我声明(须公告机构) |
| **CB** | 6-8周 | 按目标国 |
| **EMC** | 4-6周 | ❌ |
| **FCC** | 3-5周 | ❌ |

### 6.2 超节点认证时间线 (延长+并行)

```
Phase 1 (Week 1-6):  超节点预测试/自测
  ├─ 安规逐节点+整柜级
  ├─ EMC 整柜 (需10m暗室或OATS)
  ├─ 能效整柜 SERT测试
  ├─ 液冷安全评估
  └─ SPDM安全验证

Phase 2 (Week 7-14): 正式认证
  ├─ 安规: 先节点认证→延伸整柜
  ├─ EMC: 整柜测试 (2-3周实际+暗室调度)
  ├─ 能效: SERT整柜 (1-2周)
  ├─ 安全: SPDM逐设备+整柜
  └─ 并行CC+EMC+安全 (关键路径: 安规+EMC)

Phase 3 (Week 15-18): 整改+补测

Phase 4 (Week 19-22): 证书获取
```

**超节点认证总周期**: 约18-22周 (约为标准服务器1.5-2倍)

### 6.3 认证复用策略

| 策略 | 说明 | 节省 |
|------|------|:---:|
| **平台共用** | 同平台/结构机型共用安全结构评估 | 30-50%费用 |
| **电源预认证** | 采用已有认证PSU, 仅做整机温升+异常 | 20-30%周期 |
| **CB转多国** | 一份CB报告转CCC/BSMI/KC/PSE | 50-70%多国认证费 |
| **ODM/OEM共用** | 共用制造商已有的安全认证 | 40-60% |
| **节点先行** | 单节点认证后再延伸整柜 | 超节点策略 |
| **EMC预扫** | 内部预扫减少正式测试整改 | 30-50%补测时间 |

## 十、2025-2027 认证合规最新法规 (v3.1 联网检索补充)

### 10.1 EU Cyber Resilience Act (CRA) 完整时间线 (来源: EU官方 2024-2027)

EU CRA (Regulation 2024/2847) 是影响服务器出口欧盟的最重要网络安全法规:

| 时间节点 | 要求 | 影响服务器厂商 |
|------|------|:---:|
| **2024-12-11** | 法规生效 | 启动合规准备 |
| **2025-06-11** | 报告义务(已知漏洞/事件) | 建立漏洞披露流程 |
| **2026-09-11** | 报告义务全面生效 | 必须报告严重漏洞+事件 |
| **2027-12-11** | **全面强制执行** | 所有在欧销售产品必须合规 |

**CRA核心要求**:
- **CE标志**: 产品须贴CE标志证明CRA合规
- **安全设计(Security by Design)**: 默认安全配置, 最小权限
- **漏洞报告**: 24小时内报告ENISA, 72小时详细报告
- **SBOM**: 强制提供软件物料清单
- **免费安全更新**: 产品生命周期内(至少5年)
- **HBOM**: 硬件物料清单(Hardware BOM), 含芯片级供应链

**违规处罚**: 最高1500万欧元或全球营收2.5%

### 10.2 SBOM 全球法规对比 (来源: NIST/CAICT 2025)

| 地区 | 法规 | 强制范围 | 生效时间 | 格式 |
|------|------|:---:|:---:|:---:|
| **美国** | EO 14028 + NIST SP 800-218 | 联邦采购软件 | 2023起 | SPDX/CycloneDX |
| **欧盟** | CRA | 所有联网产品 | 2027-12 | SPDX/CycloneDX |
| **中国** | CAICT软件物料清单规范 | 重点行业 | 2023起 | SPDX-CN |
| **日本** | 经产省指南 | 政府采购 | 2024起 | SPDX |

**SBOM内容要求**:
- 组件名称+版本+供应商
- 依赖关系(传递依赖)
- 许可证信息
- 已知漏洞(CVE关联)
- 哈希值(完整性校验)

**生成工具**: Black Duck / Snyk / Anchore / Trivy / Syft

### 10.3 SPDM 安全链最新标准 (来源: OCP/DMTF 2025)

SPDM (Security Protocol and Data Model) 1.3 + OCP DC-SCM 2.1 构成服务器安全链:

| SPDM版本 | 关键特性 | 应用层级 |
|:---:|------|:---:|
| 1.0 | 基础认证+测量 | BMC↔PSU |
| 1.1 | 会话密钥协商 | BMC↔网卡 |
| 1.2 | 互认证+证书链 | BMC↔GPU/CPU |
| **1.3** | **抗量子+密钥轮换** | **全链路** |

**OCP DC-SCM 2.1新增** (2025):
- TCU(Trust Compute Unit)集成Axiado方案, BMC+PRoT+TPM三合一
- 支持Lattice CPLD实现SCM-HPM(主机处理器模块)连接
- 抗量子算法(ML-KEM/ML-DSA)预留

### 10.4 800V HVDC 安规新要求 (来源: IEC/NVIDIA 2025)

800V HVDC引入新安规要求:

| 标准 | 范围 | 关键要求 |
|------|------|------|
| **IEC 62477-1** | 电力电子系统安全 | 爬电距离≥25mm@800V, 电气间隙≥16mm |
| **IEC 60664-1** | 绝缘配合 | 过电压类别II, 污染度2 |
| **IEC 62368-1** | 音视频IT设备安全 | 800V需附加保护接地+绝缘监测 |
| **GB/T 3805** | 特低电压限值 | 800V远超ELV, 需危险电压标识 |

**新增测试项**:
- 介质耐压: 800V系统需2500V AC耐压测试(较48V的1000V提升2.5×)
- 绝缘电阻: ≥100MΩ(较48V的10MΩ提升10×)
- 电弧检测: 800V直流电弧风险, 需AFD(Arc Fault Detection)

### 10.5 UALink 开放标准认证 (来源: UALink Consortium 2025)

UALink 1.0于2025年4月发布, 开放互联标准带来新认证维度:

**UALink合规要求**:
- 互操作性: 不同厂商GPU/交换机互通
- 一致性测试: UALink Consortium认证
- 兼容性矩阵: 已通过测试的设备组合

**与NVLink对比**:
- NVLink: NVIDIA封闭生态, 无需第三方认证
- UALink: 开放标准, 需通过Consortium认证, 类似PCIe/SIG模式

**首批UALink设备**: 2026年面世, AMD/Broadcom/Intel主导

### 13.1 TPM 2.0 可信认证 (来源: TCG/21ic 2025)

TPM 2.0是可信计算"硬件根", 服务器安全基础:

**TPM 2.0认证体系**:
- **PCR度量**: 启动链各阶段度量值
- **远程证明**: 服务端验证客户端可信
- **密封加密**: 绑定平台状态
- **密钥保护**: 硬件级密钥存储

**TPM 1.2 vs 2.0**:
| 维度 | TPM 1.2 | TPM 2.0 |
|------|:---:|:---:|
| 算法 | RSA/SHA-1 | RSA/ECC/SHA-256/SHA-384 |
| 灵活性 | 低 | 高(算法可替换) |
| 国密 | 不支持 | 兼容(SM2/SM3/SM4) |
| 发布 | 2003 | 2014 |

**国密TCM**:
- 中国国密可信平台模块
- SM2(非对称)/SM3(哈希)/SM4(对称)
- 与TPM 2.0并存, 信创要求

**可信启动链认证**:
```
CPU RoT → TPM度量 → UEFI SecureBoot → OS内核 → 应用
   ↓        ↓            ↓              ↓        ↓
硬件根    PCR0-15      签名验证       IMA      远程证明
```

### 13.2 NVMe-oF 存储安全 (来源: NVMe/联瑞 2025)

NVMe-oF存储网络安全:

**安全机制**:
- 传输加密: TLS/TCP, IPSec/RDMA
- 认证: DH-HMAC-CHAP
- 访问控制: 基于主机身份
- 数据完整性: T10-PI

**NVMe-oF认证**:
- DH-HMAC-CHAP: Diffie-Hellman + HMAC
- 双向认证: 主机↔控制器
- 密钥管理: 集中KMS

### 13.3 Redfish 安全 (来源: DMTF 2025)

Redfish安全较IPMI大幅增强:

| 维度 | IPMI | Redfish |
|------|:---:|:---:|
| 传输 | 明文/弱加密 | HTTPS/TLS 1.3 |
| 认证 | 弱密码 | OAuth 2.0/JWT |
| 授权 | 角色 | RBAC |
| 审计 | 有限 | 完整审计日志 |
| 漏洞 | 多(CVE) | 少(现代设计) |

**Redfish安全最佳实践**:
- 强制HTTPS
- 多因素认证
- RBAC权限分级
- 审计日志完整
- 定期安全扫描
