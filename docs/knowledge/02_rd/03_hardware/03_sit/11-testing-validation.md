## 1 测试验证体系总览

### 1.1 全流程验证阶段

```
架构方案设计 → 电气/结构规则检查 → 多维度预仿真(SI/PI/热/EMC)
     → 集成校验 → 仿真闭环验证 → 多专业联合评审
     → 样机打样 → 单机测试 → 集群联调测试
     → 小批量试产 → 量产固化
```

### 1.2 测试分类矩阵

| 测试类别 | 子类 | 测试内容 | 标准 |
|:---------|:-----|:---------|:-----|
| **单机测试** | 功能测试 | 所有软硬件接口连通性 | — |
| | 性能测试 | CPU/GPU 吞吐, 内存带宽 | SPEC, STREAM |
| | 可靠性测试 | 高低温/振动/冲击/湿热 | IEC 60068 |
| | SI 测试 | 224G/112G 全链路 S 参数 | BER≤1e-18 |
| | EMI 测试 | 电磁兼容 | Class B |
| **集群测试** | NCCL 测试 | AllReduce 带宽/延迟 | NCCL Bench |
| | 一致性测试 | 多节点性能一致性 | ±5% |
| | 长稳测试 | 满负载 72h+ | 无故障 |
| | 容错测试 | 节点/链路故障 → 自动恢复 | — |

## 2 兼容性测试规范

### 2.1 测试范围

| 维度 | 覆盖内容 |
|:-----|:---------|
| **硬件兼容性** | 历史卡 (PCIe 3.0/4.0)、历史存储 (SATA/NVMe)、历史电源、新旧 DPU/网卡混插 |
| **接口兼容性** | BIOS/BMC/CPLD/PMC 与各硬件 I2C/SPI/GPIO/PCIe 接口 |
| **散热兼容性** | 风冷/液冷模式切换、历史卡散热适配、混插散热 |
| **固件兼容性** | 全固件模块协同、升级/回滚、历史设备驱动兼容 |

### 2.2 测试流程

```
测试准备 → 单设备适配测试 → 混插适配测试
   → 模式切换测试 → 固件兼容性测试 → 可靠性测试 → 测试总结
```

### 2.3 核心测试标准

| 测试项 | 通过标准 |
|:-------|:---------|
| 设备识别 | BIOS/BMC 100% 正常识别, 无误识别 |
| 混插稳定 | 新旧设备无资源冲突, 无性能下降 |
| 接口交互 | 连通性良好, 无丢包, 带内/带外功能正常 |
| 散热 | 各部件温度在规定阈值内 |
| 固件兼容 | 升级/回滚后功能正常, 配置保留 |

### 2.4 测试交付物

| 交付物 | 内容 |
|:-------|:------|
| 测试报告 | 范围/标准/流程/数据/结论 |
| 问题整改报告 | 整改方案/过程/结果 |
| 适配清单 | 历史/新设备型号/固件版本/注意事项 |
| 测试工具与脚本 | 可重复执行 |

## 3 设计验证准入标准

| 验证项 | 方法 | 强制标准 |
|:-------|:-----|:---------|
| 热仿真+实测 | Ansys Icepak / 6SigmaET | 芯片结温在规格内, 余量≥5℃ |
| 供电负载仿真 | PSIM / TI WEBENCH | 冗余测试全覆盖 |
| 流体仿真 | Fluent / 实测 | 无死角、无漏液 |
| EMC 仿真+实测 | CST / 暗室 | Class B 标准 |
| 振动测试 | IEC 60068-2-6 | 无松动、无异常 |
| SI 仿真+实测 | Sigrity / VNA | BER≤1e-18 |
| 风扇满速测试 | — | 风压≥50Pa |
| 漏液检测 | — | ≤1ml 触发告警 |

## 4 NCCL 集群测试方案

| 测试项 | 指令/工具 | 指标 | 通过标准 |
|:-------|:----------|:-----|:---------|
| AllReduce 带宽 | nccl_allreduce | 节点内/跨节点 | ≥理论值 90% |
| All-to-All 带宽 | nccl_alltoall | 多节点并发 | ≥理论值 85% |
| 延迟 | nccl latency | μs 级 | 在规范内 |
| 多节点一致性 | 对比各节点 | 各 GPU 性能 | ±5% |
| 长稳 | 满负载 72h | 无 NCCL 超时 | 0 故障 |
| 故障容错 | 热插拔 GPU/网卡 | 自动恢复 | 不影响业务 |
