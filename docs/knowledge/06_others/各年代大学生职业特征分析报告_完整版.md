## 一、数据说明与方法论

### 1.1 时间范围定义
| 维度 | 范围 | 说明 |
|------|------|------|
| 出生年代 | 1970年1月1日 - 2030年12月31日 | 覆盖60年人口出生跨度 |
| 毕业时间 | 出生+22岁（常规4年制本科） | 1992年 - 2052年 |
| 退休年龄 | 60岁（出生+60岁） | 2030年 - 2090年 |
| 当前分析时间 | 2026年 | 作为基准年份 |

### 1.2 核心数据指标
| 指标 | 定义 | 计算方法 |
|------|------|----------|
| 出生人数 | 该年代全国出生人口总数 | 国家统计局数据 |
| 大学入学率 | 该年代人群进入大学的比例 | 教育部数据 |
| 毕业生数量 | 该年代人群毕业当年高校毕业生总数 | 教育部历年统计 |
| 市场存量 | 当前在劳动力市场中的大学毕业生数量 | 累计毕业 - 累计退休 - 退出劳动力市场 |
| 退休数量 | 已达到退休年龄的人数 | 按60岁退休计算 |
| 净增量 | 每年新增毕业生 - 退休人数 | 劳动力市场变化量 |

### 1.3 数据来源
- **教育部**：历年高校毕业生数据、入学率统计
- **国家统计局**：人口出生数据、劳动力市场数据
- **行业报告**：知识库中各行业分析报告
- **预测模型**：基于人口结构和趋势的预测
